Plugin Manager

Plugins are essentially code snippets which when called via various parameters either activate a program, script or perform a specific function. They can be as simple as a function which replaces typed text with code (such as typing to display an image in a content item) to full fledged third party libraries (such as WYSIWYG editors or other programs).

Detailed functions for default Plugins may be found here:
Plugins: New/Edit

Plugins are divided into 4 types/categories that may be displayed by filtering in the filter field:
  • content
  • editors
  • editors-xtd
  • search


Toolbar Icons:

Select a Plugin by clicking on the checkbox next to its name.

Publish/ Unpublish: Click the icon to Publish or Unpublish the Plugin.

New:  Click the "New" icon to create a new Plugin. Then fill in the fields listed.

Edit: Click the "Edit" icon to change the settings as desired.

Delete: Click the "Delete" icon. This will uninstall it.

WARNING: This cannot be undone. Deleted Plugins cannot be restored as a "Menu Item" or "Content Item" can. If you have deleted a Plugin and you wish to use that Plugin again, you must re-Install the Plugin.
NOTE: Plugins essential to core operations cannot be deleted.


Columns:

Plugin Name : This lists the names of the Plugins currently installed. Click the name to edit any of the plugins.

Published : This shows whether the Plugin is published or unpublished. Click on the icon if you wish to toggle its state.

Reorder :  Click the arrow icons to move the item up or down in the list. This has no effect on the Front-end of your site.  You only reorder to help you organize your list of plugins in the Back-end.

Order : This shows the order of Plugins in relation to other Plugins of the same type. Type  the new numbers in the order that you wish your plugins to appear in the Back-end.
