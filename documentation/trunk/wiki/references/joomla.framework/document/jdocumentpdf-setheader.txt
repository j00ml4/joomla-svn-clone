====== setHeader ======
{#JAPI Joomla.Framework Document JDocumentPDF::setHeader #}

Sets the document header string

The header is used in the response header.

===== Syntax =====
void setHeader ( **$text** )

| **$text** | string | is a string containing the document header string. |

===== Examples =====
<code php|Example>
$pdfDoc =& JDocument::getInstance( 'pdf' );
$pdfDoc->setHeader( 'This is the header' );
echo $pdfDoc->getHeader();
</code>
might produce:
<code php|Result>
This is the header
</code>
