====== render ======
{#JAPI Joomla.Framework Document JDocumentRendererComponent::render #}

Renders a component script and returns the results as a string.

The method returns the value of the $content parameter.

===== Syntax =====
string render	( **$component**, **$params**, **$content** )

| **$component** | string | is a string containing the name of the component to render.  This parameter is not used.  This parameter is optional. |
| **$params** | array | is an associative array of parameter values.  This parameter is not used.  This parameter is optional. |
| **$content** | string | is a string containing the output that should be returned.  This parameter is optional and if omitted defaults to null. |

===== Examples =====
<code php|Example>
$document =& JFactory::getDocument();

jimport( 'joomla.document.renderer' );
jimport( 'joomla.document.html.renderer.component' );
$renderer = new JDocumentRendererComponent( $document );
echo $renderer->render( 'com_example', array(), 'This is the renderer output.' );
</code>
might produce:
<code php|Result>
This is the renderer output.
</code>
