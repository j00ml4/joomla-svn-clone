Returns the document name.

The name is returned as part of the header in the render() method.

===== Syntax =====
string getName ()

===== Examples =====
<code php|Example>
$pdfDoc =& JDocument::getInstance( 'pdf' );
$pdfDoc->setName( 'This is the document name' );
echo $pdfDoc->getName();
</code>
might produce:
<code php|Result>
This is the document name
</code>
