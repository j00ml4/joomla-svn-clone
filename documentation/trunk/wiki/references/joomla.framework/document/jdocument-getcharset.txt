====== getCharset ======
{#JAPI Joomla.Framework Document JDocument::getCharset #}

Returns the document charset encoding.
===== Syntax =====
string getCharset ()

===== Examples =====
<code php|Example>
global $mainframe;

$document = &JFactory::getDocument();
echo "Character Set: " . $document->getCharset();
</code>
Might produce:
<code php|Example>
Character Set: utf-8
</code>
