====== getCharset ======
^ API ^ Package ^ Class ^ phpDocumentor ^ Last reviewed ^ Doc status ^
| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#Document|Document]] | [[JDocument|JDocument]] | [[http://api.joomla.org/Joomla-Framework/Document/JDocument.html|JDocument->getCharset]] |

Returns the document charset encoding.
===== Syntax =====
string getCharset ()

===== Examples =====
<code php|Example>
global $mainframe;

$document = &JFactory::getDocument();
echo "Character Set: " . $document->getCharset();
</code>
Might produce:
<code php|Example>
Character Set: utf-8
</code>
