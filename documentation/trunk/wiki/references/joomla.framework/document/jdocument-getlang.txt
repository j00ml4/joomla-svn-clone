Gets the language of the current document.
===== Syntax =====
string getLang ()

===== Examples =====
This example displays the language code for the current document.
<code php|Example>
global $mainframe;

$document = &JFactory::getDocument();
echo "Language: " . $document->getLang();
</code>
Might produce:
<code php|Result>
Language: en
</code>
