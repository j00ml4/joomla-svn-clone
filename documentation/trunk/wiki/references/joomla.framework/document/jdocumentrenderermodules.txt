====== JDocumentRendererModules ======
{#JAPI Joomla.Framework Document JDocumentRendererModules #}

Class to render modules for the HTML document.  This class will render a group of modules from the specified position.

| [[JDocumentRendererModules-render|{{http://api.joomla.org/media/images/Method.png }}]] [[JDocumentRendererModules-render|render]] | Renders module scripts for the specified position and returns the results as a string. |
