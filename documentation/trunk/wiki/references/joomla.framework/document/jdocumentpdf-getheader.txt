====== getHeader ======
{#JAPI Joomla.Framework Document JDocumentPDF::getHeader #}

Returns the document header string.

The header is used in the response header.

===== Syntax =====
string getHeader ()

===== Examples =====
<code php|Example>
$pdfDoc =& JDocument::getInstance( 'pdf' );
$pdfDoc->setHeader( 'This is the header' );
echo $pdfDoc->getHeader();
</code>
might produce:
<code php|Result>
This is the header
</code>
