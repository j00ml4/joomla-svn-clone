====== getBase ======
{#JAPI Joomla.Framework Document JDocumentHTML::getBase #}

Returns the document base URL.
===== Syntax =====
string getBase ()

===== Examples =====
<code php|Example>
global $mainframe;

$document =& JFactory::getDocument();
echo "Base URL: " . $document->getBase();
</code>
might produce:
<code php|Result>
Base URL: http://127.0.0.1/joomla11/
</code>
