====== __construct ======
{#JAPI Joomla.Framework Document JDocumentRenderer::__construct #}

Constructor for JDocumentRenderer.

Stores a reference to the specified [[jdocument|JDocument]] object.

===== Syntax =====
JDocumentRenderer __construct	( **&$doc** )

| **&$doc** | [[jdocument|JDocument]] | is a reference to the [[jdocument|JDocument]] object that instantiated the renderer. |

===== Examples =====
This method is invoked by the [[jdocument-loadrenderer|JDocument->loadRenderer]] method in the following manner:
<code php|Example>
$instance = new $class($this);
</code>
where $class is a JDocumentRenderer class.
