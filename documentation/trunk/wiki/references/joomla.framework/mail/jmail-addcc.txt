====== addCC ======
{#JAPI Joomla.Framework Mail JMail::addCC #}

Use this class if you want to add one or more CC ([[http://en.wikipedia.org/wiki/Carbon_copy|Carbon Copy]]) recepients to the e-mail.

===== Syntax =====
void addBCC( **$cc** ) 

| **$cc** | mixed | Either a string or array of strings [recipients].|
