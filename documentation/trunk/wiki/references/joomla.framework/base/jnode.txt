====== JNode [WIP] ======
{#JAPI Joomla.Framework Base JNode #}

Abstract Tree Node Class.

This class is used by [[JTree]] to create a hierarchy of objects similar to the HTML or XML DOM.
The Joomla! menu modules are built on this foundation.
