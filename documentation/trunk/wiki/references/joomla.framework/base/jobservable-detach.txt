This member is used to detach an observer object.

===== Syntax =====
boolean detach	( **$observer** )
| **$observer** | JObserver | An observer object to detach. If this observer object is referenced in the observer array, then it is removed and the function returns true. Otherwise, the function returns false. |

===== Examples =====
If a handler $handler was previously attached, it could be detached in the following way:
<code php|Example>
$dispatcher =& JEventDispatcher::getInstance();
$dispatcher->detach( $handler );
</code>
