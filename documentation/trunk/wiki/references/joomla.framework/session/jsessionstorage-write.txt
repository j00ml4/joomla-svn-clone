====== write ======
{#JAPI Joomla.Framework Session JSessionStorage::write #}

Write session data to the SessionHandler backend.

===== Syntax =====
boolean write	( **$id**, **$session_data** )

| **$id** | string | The session identifier. |
| **$session_data** | string | The session data. |
