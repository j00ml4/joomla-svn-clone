====== JLanguage ======
{#JAPI Joomla.Framework I18N JLanguage #}

Languages/translation handler class. This class provides an interface to installed Joomla! languages. The Joomla! core will instantiate this class with the default system language. It is possible to create a second instance of the class with the same language, but it should be noted that any given language file will only be loaded once. Therefore, if the default system language is English, and the JLanguage class is instantiated again for the English language, the extension instantiated object will not be able to load any files that have already been loaded by the core.
