====== getLanguagePath ======
{#JAPI Joomla.Framework I18N JLanguage::getLanguagePath #}

Calculates the path to the specified language.

===== Syntax =====
string getLanguagePath	( **$basePath**, **$language** )

| **$basePath** | string | is a string containing the basepath to use. |
| **$language** | string | is a string containing the language tag. |

===== Examples =====
<code php|Example>
echo JLanguage::getLanguagePath( JPATH_BASE . DS . installation, 'it-IT' );
</code>
might produce:
<code php|Result>
C:\\webroot\\htdocs\\joomla15\\installation\\language\\it-IT\\
</code>
