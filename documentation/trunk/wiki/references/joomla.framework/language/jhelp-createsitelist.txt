====== createSiteList ======
{#JAPI Joomla.Framework I18N JHelp::createSiteList #}

This method builds a list of the help sites which can be used in a select option. The method returns an array of arrays that can be used to build the select options.
===== Syntax =====
array createSiteList ( **$pathToXml**, **$selected** )
| **$pathToXml** | string | is a string containing the path to an XML file. |
| **$selected** | string | is a string containing the language tag to select (if it exists). This parameter is optional and if omitted defaults to null. |

===== Examples =====
<code php|Example>
jimport( 'joomla.language.help' );

$helpsites	= array ();
$helpsites 	= JHelp::createSiteList('http://help.joomla.org/helpsites-11.xml', $row->helpurl);

print_r( $helpsites );
</code>
