====== getKnownLanguages ======
{#JAPI Joomla.Framework I18N JLanguage::getKnownLanguages #}

Returns a list of known languages in a specified folder. The method will search in the language folder of the specified path for a list of languages installed in that location (i.e. if the specified path is /var/www/joomla/installation, the method will look for XML files in the subfolders of the /var/www/joomla/installation/language folder). 

===== Syntax =====
array getKnownLanguages	( **$basePath** )

| **$basePath** | string | is a string specifying the basepath to use.  This parameter is optional and if omitted defaults to JPATH_BASE. |

===== Examples =====
<code php|Example>
$german = JLanguage::getInstance( 'de-DE' );
$languages = $german->getKnownLanguages( JPATH_BASE . DS . 'installation' );

foreach ($languages as $language) {
    echo $language['name'] . "<br />";
}
</code>
