====== getWinCP ======
{#JAPI Joomla.Framework I18N JLanguage::getWinCP #}

Gets the local windows code page.

===== Syntax =====
string getWinCP ()

===== Examples =====
<code php|Example>
$english = JLanguage::getInstance( 'en-GB' );
echo $english->getWinCP();
</code>
might produce:
<code php|Result>
iso-8859-1
</code>
