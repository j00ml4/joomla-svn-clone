====== getFloat ======
{#JAPI Joomla.Framework Environment JRequest::getFloat #}

Fetches and returns a given filtered variable.  The float  filter only allows digits and periods.  This is currently  only a proxy function for getVar().<p>See getVar() for more in-depth documentation on the parameters.</p>

===== Syntax =====
float getFloat	( **$name**, **$default**, **$hash** )

| **$name** | string | Variable name |
| **$default** | string | Default value if the variable does not exist |
| **$hash** | string | Where the var should come from (POST, GET, FILES, COOKIE, METHOD) |
