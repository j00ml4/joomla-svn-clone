====== getCmd ======
{#JAPI Joomla.Framework Environment JRequest::getCmd #}

Fetches and returns a given filtered variable. The cmd  filter only allows the characters [A-Za-z0-9.-_]. This is  currently only a proxy function for getVar().<p>See getVar() for more in-depth documentation on the parameters.</p>

===== Syntax =====
string getCmd	( **$name**, **$default**, **$hash** )

| **$name** | string | Variable name |
| **$default** | string | Default value if the variable does not exist |
| **$hash** | string | Where the var should come from (POST, GET, FILES, COOKIE, METHOD) |
