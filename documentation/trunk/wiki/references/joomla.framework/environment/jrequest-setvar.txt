====== setVar ======
{#JAPI Joomla.Framework Environment JRequest::setVar #}

Set a variabe in on of the request variables


===== Syntax =====
string setVar	( **$name**, **$value**, **$hash**, **$overwrite** )

| **$name** | string | the name of the variable to set |
| **$value** | string | the value to be inserted for the specified variable |
| **$hash** | string | the hast to insert into (post, get, ...) |
| **$overwrite** | boolean | true to overwrite any existing variable with the same name, false to set only if not already set |
