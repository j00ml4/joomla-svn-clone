====== getDBO ======

| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#Application|Application]] | {{http://api.joomla.org/media/images/Class.png?nolink}}[[JApplication|JApplication]] | [[http://api.joomla.org/Joomla-Framework/Application/JApplication.html|JApplication->getDBO]] | never | Work in progress |

Return a reference to a JDatabase object instance. This method is the means by which extensions should obtain access to the Joomla! database object.

===== Syntax =====
JDatabase &getDBO ()

===== Examples =====
<code php|Example>
global $mainframe;

$database = $mainframe->getDBO();    
</code>
will result in a reference to the application database object being stored in the $database variable.
