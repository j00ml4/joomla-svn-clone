====== getCustomPathWay ======

Returns an array of names of pathway items. This method is deprecated in Joomla! 1.5 (JPathWay->getNamePathWay should be used instead).

===== Syntax =====
void getCustomPathWay ()


===== Examples =====
<code php|Example>
global $mainframe;

$mainframe->appendPathWay( "CustomPathItem1", "http://www.test1.ca" );
$mainframe->appendPathWay( "CustomPathItem2", "http://www.test2.ca" );
print_r ( $mainframe->getCustomPathWay () );
</code>
might produce
<code php|Result>
Array
(
    [0] => 
    [1] => Home
    [2] => contact
    [3] => CustomPathItem1
    [4] => CustomPathItem2
)
</code>
