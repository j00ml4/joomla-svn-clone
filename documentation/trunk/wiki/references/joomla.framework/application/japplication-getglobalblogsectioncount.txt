====== getGlobalBlogSectionCount ======
{#JAPI Joomla.Framework Application JApplication::getGlobalBlogSectionCount #}

Returns the number of items in the menu that are of type 'global blog section'. This method is deprecated in Joomla! 1.5 (JApplicationHelper->getItemCount should be used instead).

===== Syntax =====
void getGlobalBlogSectionCount ()

===== Examples =====
<code php|Example>
global $mainframe;

echo $mainframe->getGlobalBlogSectionCount();
</code>
will display the number of menu items that are of type 'global blog section'.
