====== _ ======
{#JAPI Joomla.Framework Application JRoute::_ #}

Translates an internal Joomla URL to a humanly readible URL.

===== Syntax =====
The _	( **$url**, **$xhtml**, **$ssl** )

| **$url** | string | Absolute or Relative URI to Joomla resource |
| **$xhtml** | boolean | Replace &amp; by &amp;amp; for xml compilance |
| **$ssl** | int | Secure state for the resolved URI           1: Make URI secure using global secure site URI           0: Leave URI in the same secure state as it was passed to the function          -1: Make URI unsecure using the global unsecure site URI |
