====== getDocument ======
{#JAPI Joomla.Framework Application JApplication::getDocument #}

Return a reference to the JDocument object. This method is the means by which extensions should obtain access to the Joomla! document object, which can be used to set and retrieve information about the current document. See the [[references:joomla.framework:document:jdocument|JDocument]] class for more information.

===== Syntax =====
void &getDocument ( **$type** )

| **$type** | string | is a string containing the type of document object to be retrieved. This parameter is optional and if omitted defaults to 'html'. |

===== Examples =====
<code php|Example>
global $mainframe;

$document = JFactory::getDocument();
</code>
will result in a reference to the document object being stored in the $document variable.
