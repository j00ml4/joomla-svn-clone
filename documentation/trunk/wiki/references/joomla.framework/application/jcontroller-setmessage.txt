====== setMessage ======
{#JAPI Joomla.Framework Application JController::setMessage #}

Sets the internal message that is passed with a redirect.

This method is displayed at the top of the target page.

===== Syntax =====
void setMessage	( **$text** )
| **$text** | string | is a string containing the message that is to be displayed.|


===== Examples =====
<code php|Example>
if (JRequest::getVar( 'testvalue' ) != 1) {
	$controller->setRedirect( 'index.php?option=com_weblinks&testvalue=1&view=weblinks' );
	$controller->setMessage( 'This is the redirect message!' );
	$controller->redirect();
}
</code>
will result in the browser being redirected and the specified message being displayed at the top of the target page.
