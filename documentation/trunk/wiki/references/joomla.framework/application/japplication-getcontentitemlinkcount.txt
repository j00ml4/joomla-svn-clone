====== getContentItemLinkCount ======
{#JAPI Joomla.Framework Application JApplication::getContentItemLinkCount #}

Returns the number of items in the menu that are of type 'content item link'.  This method is deprecated in Joomla! 1.5 (JApplicationHelper->getItemCount should be used instead).

===== Syntax =====
void getContentItemLinkCount ()

===== Examples =====
<code php|Example>
global $mainframe;

echo $mainframe->getContentItemLinkCount();
</code>
will display the number of menu items that are of type 'content item link'.
