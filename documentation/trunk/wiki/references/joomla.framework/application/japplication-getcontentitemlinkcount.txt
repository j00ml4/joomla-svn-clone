====== getContentItemLinkCount ======
{#JAPI Joomla.Framework Application JApplication::getContentItemLinkCount - IR #}

Returns the number of items in the menu that are of type 'content item link'.

**The method is deprecated** and provided for backwards compatibility with Joomla! 1.0. It is scheduled to be removed in future versions.
Use [[JApplicationHelper-getItemCount|JApplicationHelper::getItemCount]] instead.

===== Syntax =====
void getContentItemLinkCount ()

===== Examples =====
<code php|Example>
global $mainframe;

echo $mainframe->getContentItemLinkCount();
</code>
will display the number of menu items that are of type 'content item link'.
