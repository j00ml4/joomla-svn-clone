====== getLayout ======
{#JAPI Joomla.Framework Application JView::getLayout #}

Gets the current layout.

Returns a string containing the current layout.

===== Syntax =====
string getLayout ()


===== Examples =====
<code php|Example>
echo $view->getLayout();
</code>
might produce:
<code php|Result>
default
</code>
