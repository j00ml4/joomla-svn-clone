====== getComponent ======
{#JAPI Joomla.Framework Application JComponentHelper::getComponent #}

Gets information about a component.

During the first invokation of this method, all the entries are retrieved from the components table. Subsequent calls will return the record for the specified component.

===== Syntax =====
stdClass &getComponent ( **$name**, **$strict** )
| **$name** | string | The name of the component for which to obtain information. |
| **$strict** | bool | If set and a component does not exist, the enabled attribue will be set to false |

The value of %%$name%% is equal to the %%option%% column.

===== Examples =====

<code php|Example>
$info =& JComponentHelper::getComponent( 'com_weblinks' );
print_r( $info );
</code>
might produce
<code php|Result>
stdClass Object
(
    [id] => 4
    [name] => Web Links
    [link] => option=com_weblinks
    [menuid] => 0
    [parent] => 0
    [admin_menu_link] => 
    [admin_menu_alt] => Manage Weblinks
    [option] => com_weblinks
    [ordering] => 0
    [admin_menu_img] => js/ThemeOffice/component.png
    [iscore] => 0
    [params] => 
    [enabled] => 1
)
</code>
