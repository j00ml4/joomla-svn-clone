====== getBlogCategoryCount ======
{#JAPI Joomla.Framework Application JApplication::getBlogCategoryCount #}

Returns the number of items in the menu that are of type 'blog category'.  This method is deprecated in Joomla! 1.5 (JApplicationHelper->getItemCount should be used instead).

===== Syntax =====
void getBlogCategoryCount ()


===== Examples =====
<code php|Example>
global $mainframe;

echo $mainframe->getBlogCategoryCount();
</code>
will display the number of menu items that are of type 'blog category'.
