====== getInstance ======
{#JAPI Joomla.Framework Application JMenu::getInstance #}

Returns a reference to the global JMenu object, only creating it if it  doesn't already exist.<p>This method must be invoked as:  		<pre>  $menu = &JMenu::getInstance();</pre></p>

===== Syntax =====
<a href="../../Joomla-Framework/Application/JMenu.html">JMenu</a> &getInstance ()
