====== getHead ======
{#JAPI Joomla.Framework Application JApplication::getHead - IR #}

Returns the head block of the current document. 

**The method is deprecated** and provided for backwards compatibility with Joomla! 1.0. It is scheduled to be removed in future versions.
Use [[references:joomla.framework:document:JDocument-renderHead|JDocument::renderHead]] instead.

===== Syntax =====
void getHead ()

===== Examples =====
<code php|Example>
global $mainframe;

echo $mainframe->getHead();
</code>
might produce
<code php|Result>
this method was broken at time of writing.
</code>
