====== getHead ======
{#JAPI Joomla.Framework Application JApplication::getHead #}

Returns the head block of the current document. This method is deprecated in Joomla! 1.5 (JDocument->renderHead should be used instead).

===== Syntax =====
void getHead ()

===== Examples =====
<code php|Example>
global $mainframe;

echo $mainframe->getHead();
</code>
might produce
<code php|Result>
this method was broken at time of writing.
</code>
