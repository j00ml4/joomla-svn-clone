====== getBrowser ======
{#JAPI Joomla.Framework Application JApplication::getBrowser #}

Returns a reference to the [[references:joomla.framework:environment:jbrowser|JBrowser]] object. This method allows Joomla! extensions to access information about the client browser.

===== Syntax =====
JBrowser &getBrowser ()

===== Examples =====
<code php|Example>
global $mainframe;

$browser = $mainframe->getBrowser();
</code>
will result in $browser containing a reference to the application's browser object.
