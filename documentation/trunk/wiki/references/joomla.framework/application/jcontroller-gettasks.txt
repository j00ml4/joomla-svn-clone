====== getTasks ======
{#JAPI Joomla.Framework Application JController::getTasks #}

Gets the available tasks in the controller.

===== Syntax =====
array getTasks ()

===== Examples =====
<code php|Example>
$controller = new JController();
print_r( $controller->getTasks() );
</code>
might produce
<code php|Result>
Array ( [0] => display )
</code>
