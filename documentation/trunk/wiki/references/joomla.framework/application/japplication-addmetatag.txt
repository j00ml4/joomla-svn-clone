====== addMetaTag ======

Adds a meta tag to the head block of the current document. This method is deprecated in Joomla! 1.5 (JDocument->setMetaData should be used instead). The **$prepend** and **$append** parameters have no effect in Joomla! 1.5.

===== Syntax =====
void addMetaTag	( **$name**, **$content**, **$prepend**, **$append** )

| **$name** | string | is a string containing the name of the meta tag to add to the document. |
| **$content** | string | is a string containing the content to add to the meta tag. |
| **$prepend** | string | is not used in Joomla! 1.5. |
| **$append** | string | is not used in Joomla! 1.5. |

===== Examples =====
<code php|Example>
global $mainframe;

$mainframe->addMetaTag( "metatagname", "metatagcontent" );
</code>
will insert the following meta tag into the head block of the current document:
<code php|Result>
<meta name="metatagname" content="metatagcontent" />
</code>
