====== getOption ======

| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#Application|Application]] | {{http://api.joomla.org/media/images/Class.png?nolink}}[[JApplication|JApplication]] | [[http://api.joomla.org/Joomla-Framework/Application/JApplication.html|JApplication->getOption]] | never | Work in progress |

Return the application option string. The option string contains the name of the main page component.

===== Syntax =====
string getOption ()

===== Examples =====
<code php|Example>
global $mainframe;

echo $mainframe->getOption();
</code>
might produce:
<code php|Result>
com_contact
</code>
if the contact component was currently being executed.
