====== appendMetaTag ======
{#JAPI Joomla.Framework Application JApplication::appendMetaTag - IR #}

Adds a meta tag to the head block of the current document. 

**The method is deprecated** and provided for backwards compatibility with Joomla! 1.0. It is scheduled to be removed in future versions.
Use [[references:joomla.framework:document:JDocument-setMetaData|JDocument::setMetaData]] instead.

===== Syntax =====
void appendMetaTag	( **$name**, **$content** )

| **$name** | string | the name of the meta tag to add to the document. |
| **$content** | string | the content to add to the meta tag. |

===== Examples =====
<code php|Example>
global $mainframe;

$mainframe->appendMetaTag( "metatagname", "metatagcontent" );
</code>
will insert the following meta tag into the head block of the current document:
<code php|Result>
<meta name="metatagname" content="metatagcontent" />
</code>
