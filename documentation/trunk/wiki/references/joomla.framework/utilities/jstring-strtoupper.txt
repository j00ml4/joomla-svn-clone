====== strtoupper ======
{#JAPI Joomla.Framework Utilities JString::strtoupper #}

Makes a string uppercase. Note: The concept of a characters "case" only exists in some alphabets such as Latin, Greek, Cyrillic, Armenian and archaic Georgian - it does  not exist in the Chinese alphabet, for example. See [[http://www.unicode.org/reports/tr21/tr21-5.html|Unicode Standard Annex #21: Case Mappings]]. See [[http://php.net/manual/function.strtoupper.php|strtoupper]] for more information.

===== Syntax =====
string strtoupper ( **$str** )

| **$str** | string | is the string to be converted to upper case. |

===== Examples =====
<code php|Example>
$string = "Και Εισηλθεν Παλιν Εις";
echo "The string $string converted to upper case is " . JString::strtoupper( $string ) . ".";
</code>
might produce:
<code php|Result>
The string Και Εισηλθεν Παλιν Εις converted to upper case is ΚΑΙ ΕΙΣΗΛΘΕΝ ΠΑΛΙΝ ΕΙΣ.
</code>
