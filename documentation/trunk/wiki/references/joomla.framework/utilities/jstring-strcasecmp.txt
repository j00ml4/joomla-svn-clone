====== strcasecmp ======
{#JAPI Joomla.Framework Utilities JString::strcasecmp #}

Compares two strings in a case-insensitive manner. This method takes into account different collations that various character sets might have. See [[http://php.net/manual/function.strcasecmp.php|strcasecmp]] for more information.

===== Syntax =====
integer strcasecmp ( **$str1**, **$str2** )

| **$str1** | string | is a string containing the first string to compare. |
| **$str2** | string | is a string containing the second string to compare. |

===== Examples =====
<code php|Example>
$string1 = "οἱ γὰρ";
$string2 = JString::strtoupper( $string1 );
echo "$string1 and $string2 are " . (JString::strcasecmp( $string1, $string2 ) ? "not equal" : "equal");
echo " when compared without regard to case.";
</code>
might produce:

</code>
