====== JBufferStream ======
{#JAPI Joomla.Framework Utilities JBufferStream #}

Generic Buffer stream handler<p>This class provides a generic buffer stream.  It can be used to store/retrieve/manipulate string buffers with the standard  PHP filesystem I/O methods.</p>
