====== sortObjects ======
{#JAPI Joomla.Framework Utilities JArrayHelper::sortObject #}

Sorts an array of objects on a given field. This method works directly on the passed array.

===== Syntax =====
array sortObjects ( **&$a**, **$k**, **$direction** )

| **$arr** | array | is an array containing the objects to be sorted. |
| **$k** | string | is a string containing the name of the key on which to sort. |
| **$direction** | integer | is an integer containing the direction in which to sort (1 = Ascending, -1 = Descending). This parameter is optional and if omitted defaults to 1 (ascending). |

===== Examples =====
<code php|Example>
$db =& JFactory::getDBO();
 
$query = "SELECT `id`, `title`, `title_alias`"
. "\n   FROM #__content"
;
$db->setQuery( $query );
if (!$results = $db->loadObjectList( 'id' )) {
    echo $db->getErrorMsg();
}

echo "<p>Normal Order:<br />";
foreach($results as $result) {
    echo $result->id . " " . $result->title . " " . $result->title_alias . "<br />";
}
echo "</p>";

JArrayHelper::sortObjects( $results, 'title' );

echo "<p>Ascending Order:<br />";
foreach($results as $result) {
    echo $result->id . " " . $result->title . " " . $result->title_alias . "<br />";
}
echo "</p>";

JArrayHelper::sortObjects( $results, 'title', -1 );

echo "<p>Descending Order:<br />";
foreach($results as $result) {
    echo $result->id . " " . $result->title . " " . $result->title_alias . "<br />";
}
echo "</p>";
</code>
