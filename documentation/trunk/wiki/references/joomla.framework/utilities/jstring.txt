====== JString ======
{#JAPI Joomla.Framework Utilities JString #}

This class handles strings that contain UTF-8 data.  It is a wrapper for the PHPUTF8 library.  All functions assume the validitry of the UTF-8 strings that are being operated on.

===== Methods =====

See [[http://php.net/manual/function.ltrim.php|ltrim]] for more information. |
