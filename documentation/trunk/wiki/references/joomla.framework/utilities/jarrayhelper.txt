====== JArrayHelper ======
{#JAPI Joomla.Framework Utilities JArrayHelper #}

JArrayHelper is an array utility class for performing various operations to work with arrays.

| {{references:Method.png?nolink}} [[JArrayHelper-sortObjects|sortObjects]] | Sorts an array of objects on a given field. |
