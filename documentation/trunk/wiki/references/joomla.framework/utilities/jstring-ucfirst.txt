====== ucfirst ======
{#JAPI Joomla.Framework Utilities JString::ucfirst #}

Makes a string's first character uppercase. See [[http://php.net/manual/function.ucfirst.php|ucfirst]] for more information.

===== Syntax =====
string ucfirst ( **$str** )

| **$string** | string | is the string to convert to sentence case (the first letter capitalized). |

===== Examples =====
<code php|Example>
$string = "και εισηλθεν παλιν εις";
echo "If we ucfirst $string we get: " . JString::ucfirst( $string ) . ".";
</code>
might produce:
<code php|Result>
If we ucfirst και εισηλθεν παλιν εις we get: Και εισηλθεν παλιν εις.
</code>
