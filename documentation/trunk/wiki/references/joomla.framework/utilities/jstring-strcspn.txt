====== strcspn ======

Finds the initial length of the string that does not contain certain characters. See [[http://php.net/manual/function.strcspn.php|strcspn]] for more information.

===== Syntax =====
integer strcspn	( **$str**, **$mask**, **$start**, **$length** )

| **$str** | string | is the string to search for the mask string. |
| **$mask** | string | is a string containing the mask characters. |
| **$start** | integer | is an integer containing the optional starting character position (in characters). |
| **$length** | integer | is an integer containing the optional length of the string to search. |

===== Examples =====
<code php|Example>
$string = "και εισηλθεν παλιν εις";
$mask1 = "ε";
echo $string . "<br />$mask1<br />";
echo "There are " . JString::strcspn( $string, $mask1 ) . " characters from the beginning of $string that do not contain $mask1.";
</code>
might produce:
<code php|Result>
και εισηλθεν παλιν εις
ε
There are 4 characters from the beginning of και εισηλθεν παλιν εις that do not contain ε.
</code>
