====== strrev ======
{#JAPI Joomla.Framework Utilities JString::strrev #}

Reverses a string. See [[http://php.net/manual/function.strrev.php|strrev]] for more information.

===== Syntax =====
string strrev	( **$str** )

| **$str** | string | is the string to be reversed. |

===== Examples =====
<code php|Example>
$string = "και εισηλθεν παλιν εις";
echo "The reverse of $string is " . JString::strrev( $string ) . ".";
</code>
might produce:
<code php|Result>
The reverse of και εισηλθεν παλιν εις is ςιε νιλαπ νεθλησιε ιακ.
</code>
