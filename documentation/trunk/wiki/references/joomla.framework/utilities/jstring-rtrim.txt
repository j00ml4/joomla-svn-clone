====== rtrim ======

Strips whitespace (or other characters) from the end of the string. This method is only necessary if the optional character list argument is supplied and contains UTF-8 characters. Otherwise, the standard function will work on a UTF-8 string. See [[http://php.net/manual/function.rtrim.php|rtrim]] for more information.

===== Syntax =====
string rtrim ( **$str**, **$charlist** )

| **$str** | string | is the string to be trimmed. |
| **$charlist** | string | is a string containing the optional character list of additional characters to trim. |

===== Examples =====
The following example will strip off the taus from the beginning of the string.

<code php|Example>
$exampleGreek = "και παντες οι συμπαραγενομενοι οχλοι επι την θεωριαν ταυτην θεωρησαντες τα γενομενα τυπτοντες τα στηθη υπεστρεφονττττ";
$charList = "τ";
echo "Before: $exampleGreek<br />";
echo "Trimming off $charList<br />";
echo "The trimmed string is: " . JString::rtrim( $exampleGreek, $charList ) . ".";
</code>
might produce:
<code php|Result>
Before: και παντες οι συμπαραγενομενοι οχλοι επι την θεωριαν ταυτην θεωρησαντες τα γενομενα τυπτοντες τα στηθη υπεστρεφονττττ
Trimming off τ
The trimmed string is: και παντες οι συμπαραγενομενοι οχλοι επι την θεωριαν ταυτην θεωρησαντες τα γενομενα τυπτοντες τα στηθη υπεστρεφον.
</code>
