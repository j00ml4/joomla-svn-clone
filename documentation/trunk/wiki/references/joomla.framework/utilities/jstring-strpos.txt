====== strpos ======
{#JAPI Joomla.Framework Utilities JString::strpos #}

Finds the position of the first occurrence of a string. See [[http://php.net/manual/function.strpos.php|strpos]] for more information.

===== Syntax =====
mixed strpos	( **$haystack**, **$needle**, **$offset** )

| **$haystack** | string | is a string containing the string being examined. |
| **$needle** | string | is a string containing the string being searched for. |
| **$offset** | integer | is an integer that specifies the position from which the search should be performed. This parameter is optional and if omitted defaults to 0. |

===== Examples =====
<code php|Example>
$haystack = "και εισηλθεν παλιν εις";
$needle = "πα";
echo "The position of the first occurence of $needle in $haystack is " . JString::strpos( $haystack, $needle ) . ".";
</code>
might produce:
<code php|Result>
The position of the first occurence of πα in και εισηλθεν παλιν εις is 13.
</code>
