====== transcode ======
{#JAPI Joomla.Framework Utilities JString::transcode #}

Transcode a string from one character encoding to another.

===== Syntax =====
string transcode	( **$source**, **$from_encoding**, **$to_encoding** )

| **$source** | string | The string to transcode. |
| **$from_encoding** | string | The source encoding. |
| **$to_encoding** | string | The target encoding. |

===== Examples =====
<code php|Example>
$string = "και εισηλθεν παλιν εις";
echo "Using transcode on $string yields: ";
echo JString::transcode( $string, "UTF-8", "ISO-8859-7" );
</code>
might produce:
<code php|Result>
Using transcode on και εισηλθεν παλιν εις yields: και εισηλθεν παλιν εις
</code>    
Due to documentation system limitations, the results of this example cannot be properly displayed. If one executes this example, the first or the second version of the string will be displayed properly depending on whether UTF-8 encoding or ISO-8859-7 encoding is used.
