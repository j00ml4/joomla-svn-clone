===== Syntax =====
string getLongVersion()


===== Examples =====
Retrieve the long version number of Joomla!

<code php|Example>
$version =& new JVersion;
$long_version = $version->getLongVersion();
echo "Powered by: ". $long_version;
</code>

might produce:

<code php|Result>
Powered by: Joomla! 1.5.0 BETA [ Khepri ] 12-OCT-2006 00:00 GMT
</code>
