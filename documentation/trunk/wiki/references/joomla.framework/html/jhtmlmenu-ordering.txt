====== ordering ======

Builds the select list for menu ordering.

This method generates a select list that contains the list of menu items that are on the same level (have the same parent) and menu as the specified item.  If the $id parameter passed is zero, the row will be treated as a new row and a message will be returned instead stating that new items are always placed last in the list.

The $row object must have the required properties for the method. The required properties are: menutype, parent and ordering.

This method returns a string containing the generated select list or a string containing the hidden input field and the new item message.

===== Syntax =====
string ordering ( **&$row**, **$id** )

| **&$row** | object | is an object containing the row for which the select list will be created. It must have menutype, parent and ordering properties. |
| **$id** | integer | is an integer containing the ID of the row. If this is zero, the row will be treated as a new row. Otherwise, it will be treated as an existing row. |

===== Examples =====
<code php|Example>
$db =& JFactory::getDBO();

$query = 'SELECT * FROM #__menu';
$db->setQuery( $query );
$row = $db->loadObject();

echo JHTML::_( 'menu.ordering', $row, $row->id );
</code>
