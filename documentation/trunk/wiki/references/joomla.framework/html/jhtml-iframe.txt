====== iframe ======
{#JAPI Joomla.Framework HTML JHTML::iframe #}

Returns a string containing the code for an inline frame element.

===== Syntax =====
string iframe	( **$url**, **$name**, **$attribs** )

| **$url** | string | is a string containing the relative URL to use for the src attribute. |
| **$name** | string | is a string containing the name of the inline frame. |
| **$attribs** | array | is an associative array of attributes to add to the element. This parameter is optional and if omitted no additional attributes will be added. |

===== Examples =====
<code php|Example>
$attribs = array( 'id' => 'iframe-id', 'class' => 'frame-class' );
echo JHTML::_( 'iframe', 'http://www.joomla.org', 'Iframe-name', $attribs );
</code>
might produce:
<code php|Result>
<iframe src="http://www.joomla.org" id=iframe-id class=frame-class name="Iframe-name"></iframe>
</code>
