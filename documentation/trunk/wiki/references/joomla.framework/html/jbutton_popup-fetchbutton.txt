====== fetchButton ======
^ API ^ Package ^ Class ^ phpDocumentor ^ Last reviewed ^ Doc status ^
| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#HTML|HTML]] | {{http://api.joomla.org/media/images/Class.png?nolink}}[[JButton_Popup|JButton_Popup]] | [[http://api.joomla.org/Joomla-Framework/HTML/JButton_Popup.html#fetchbutton|JButton_Popup->fetchButton]] | never | Internal review |

Returns the HTML code for the button. The button will include a <div> cell whose class is determined by the [[JButton-fetchIconClass|fetchIconClass]] method called with $name as its parameter. There are classes defined for certain values of $name in the standard administrator template. See the [[icon_class_list|list of icon classes]] for more details. If this class is used in the frontend, the appropriate class should be defined and any images used should be included.

===== Syntax =====
void fetchButton	( **$type**, **$name**, **$text**, **$url**, **$width**, **$height**, **$top**, **$left** )

| **$type** | string | is a string containing the button type. This parameter should be 'Popup'. This parameter is optional and if omitted defaults to 'Popup'. |
| **$name** | string | is a string containing the name of the button. This parameter is used to determine the div class and the button ID. |
| **$text** | string | is a string containing the text that should be displayed on the button. This parameter is optional and if omitted defaults to "". |
| **$url** | string | is a string containing the URL to display in the popup window. This parameter is optional and if omitted defaults to "". |
| **$width** | integer | is an integer specifying the width of the popup window. This parameter is optional and if omitted defaults to 640. |
| **$height** | integer | is an integer specifying the height of the popup window. This parameter is optional and if omitted defaults to 480. |
| **$top** | integer | is an integer specifying the position of the top of the popup window. This parameter is optional and if omitted defaults to 0. |
| **$left** | integer | is an integer specifying the position of the left side of the popup window. This parameter is optional and if omitted defaults to 0. |

===== Examples =====
<code php|Example>
$bar =& new JToolBar( 'My ToolBar' );
$button =& $bar->loadButtonType( 'Popup' );
echo $button->fetchButton( 'Popup', 'new', 'Popup Test', 'http://www.joomla.org', 300, 300, 150, 150 );
</code>
might produce:
<code php|Result>
<a onclick="document.popup.show('http://www.joomla.org', 300, 300, null)">
<div class="icon-32-new" title="Popup Test" type="Popup">
</div>
Popup Test
</a>
</code>
