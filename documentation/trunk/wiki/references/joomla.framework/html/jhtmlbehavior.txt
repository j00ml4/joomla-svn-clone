====== JHTMLBehavior ======
{#JAPI Joomla.Framework HTML JHTMLBehavior #}

The JHTMLBehavior class is a utility class for Javascript behaviors.

===== Methods =====
^ Method ^ Description ^
| [[JHTMLBehavior-calendar|##http://api.joomla.org/media/images/Method.png qq]] [[JHTMLBehavior-calendar|calendar]] | Injects the necessary Javascript into the page for the date selector to function. |

| [[JHTMLBehavior-tooltip|##http://api.joomla.org/media/images/Method.png qq]] [[JHTMLBehavior-tooltip|tooltip]] | Injects the necessary Javascript into the page to display tooltips. |
