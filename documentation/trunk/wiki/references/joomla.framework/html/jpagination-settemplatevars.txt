====== setTemplateVars ======
{#JAPI Joomla.Framework HTML JPagination::setTemplateVars #}

Sets the variables for the page navigation template. This method will add three variables to the specified template: 'PAGE_LINKS', 'PAGE_LIST_OPTIONS' and 'PAGE_COUNTER'. 'PAGE_LINKS' will be assigned the result of $this->getPagesLinks( $link ). 'PAGE_LIST_OPTIONS' will be assigned the result of $this->getLimitBox( $link ). 'PAGE_COUNTER' will be assigned the result of $this->getPagesCounter().

===== Syntax =====
boolean setTemplateVars	( **&$tmpl**, **$name**, **$link** )

| **&$tmpl** | JTemplate | is a JTemplate object to which to add the pagination footer template. |
| **$name** | string | is a string containing the name of the template to which to add the pagination footer variables. This parameter is optional and if omitted defaults to 'admin-list-footer'. |
| **$link** | string | is a string containing the link parameter to pass to the [[JPagination-getPagesLinks|getPagesLinks]] and [[JPagination-getLimitBox|getLimitBox]] methods. This parameter is optional and if omitted defaults to null. |

===== Examples =====
<code php|Example>
global $mainframe, $option;
$db =& JFactory::getDBO();

$limit		= $mainframe->getUserStateFromRequest("$option.limit", 'limit', $mainframe->getCfg('list_limit'), 0);
$limitstart	= JRequest::getVar('limitstart', 0, '', 'int');
$query = "SELECT count(*) "
. "\n FROM #__weblinks"
;
$db->setQuery( $query );
$total = $db->loadResult();

jimport('joomla.presentation.pagination');
$pageNav = new JPagination( $total, $limitstart, $limit );

$query = "SELECT id, title, url, ordering "
. "\n FROM #__weblinks "
. "\n ORDER BY ordering "
;
$db->setQuery( $query, $pageNav->limitstart, $pageNav->limit );
$rows = $db->loadObjectList();

jimport( 'joomla.template.template' );
$template = new JTemplate();
$pageNav->setTemplateVars( $template );
echo $template->getVar( 'admin-list-footer', 'page_links' );
</code>
