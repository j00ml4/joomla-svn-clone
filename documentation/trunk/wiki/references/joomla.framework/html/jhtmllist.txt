====== JHTMLList ======

JHTMLList is a drawing utility class that facilities the use of various select lists.

| [[JHTMLList-section|{{references:Method.png?nolink}}]] [[JHTMLList-section|section]] | Builds the select list of active sections. |
| [[JHTMLList-specificordering|{{references:Method.png?nolink}}]] [[JHTMLList-specificordering|specificordering]] | Builds the select list for ordering of a specified table. |
