====== site ======
{#JAPI Joomla.Framework HTML JHTMLImage::site #}

Generates image tags for images contained in the site application directory.

The method first checks to see if the specified image is a parameter image (i.e. $altFile is not empty).  If it is, then the image path is returned as $altFolder . $altFile.

If the image is not a parameter image, the method will search for specified $file in the current template images directory.  If this file exists, then this image is used for the image tag.

If none of these conditions are met, the path $folder . $file is used for the image tag.

An initial slash at the beginning of the path will be removed in an attempt to make the path relative.

The method returns a string containing the path to the image (if $asTag is true), or the HTML code for an image tag for the specified image.

===== Syntax =====
void site ( **$file**, **$folder** = '/images/M_images/', **$altFile** = NULL , **$altFolder** = '/images/M_images/', **$alt** = NULL, **$attribs** = null, **$asTag** = 1)

| $file			| string | the image filename |
| $folder		| string | the image folder location (URL) |
| $altFile		| mixed  | alternative filename or -1 if part of a parameter select list |
| $altFolder	| string | the image folder location of the alternative filename (URL) |
| $alt			| string | text for the alt attribute  | 
| $attribs		| array  | assocative array for the HTML attributes of the img element |
| $asTag		| string | whether to return the IMG element (1) or it's src attribute only (0) |

===== Examples =====
<code php|Example>
echo JHTML::_( 'image.site', 'edit.png' );
echo JHTML::_( 'image.site', 'edit.png', '/images/M_images/', 'save.png', '/images/', 'Save' );
echo JHTML::_( 'image.site', 'edit', '/images/M_images/', 'save.png', '/images/', 'Save', null, false );
</code>
might produce:
<code php|Result>
<img src="images/M_images/edit.png" alt="" align="top" border="0" />
<img src="images/save.png" alt="Save" align="top" border="0" />
images/save.png
</code>
