====== images ======

Builds the select list to choose an image.

The method will search through the specified directory for a list of image files (with extensions bmp, jpg, png or gif) and will create a select box, allowing the user to choose one of these images.

The method returns the generated HTML code for the select list.

===== Syntax =====
string images	( **$name**, **&$active**, **$javascript**, **$directory** )

| **$name** | string | is a string containing the name attribute to be used for the select list. |
| **&$active** | string | is a string containing the name of the image that is currently selected. |
| **$javascript** | string | is a string containing any Javascript that should be added to the select box. This parameter is optional and if omitted defaults to null. |
| **$directory** | string | is a string containing the directory in which to search for images. This parameter is optional and if omitted the /images/stories directory will be used. |

===== Examples =====
<code php|Example>
echo JHTML::_( 'list.images', 'image', 'key.jpg' );
</code>
might produce:
