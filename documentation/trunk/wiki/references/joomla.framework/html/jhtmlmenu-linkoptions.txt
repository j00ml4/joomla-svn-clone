====== linkoptions ======

Builds the multiple select list options for menu links/pages.

This method invokes the treerecurse method to retrieve the list of items and creates list options from this list.  This method could be used for creating a list of pages that modules to which modules could be attached.

This method returns an array of select list options.

===== Syntax =====
array linkoptions ( **$all**, **$unassigned** )

| **$all** | boolean | is a boolean value specifying whether or not an 'All' option should be added to the group. This parameter is optional and if omitted defaults to false. |
| **$unassigned** | boolean | is a boolean value specifying whether or not an 'Unassigned' option should be added to the group. This parameter is optional and if omitted defaults to false. |

===== Examples =====

$options = JHTML::_( 'menu.linkoptions' );
print_r( $options );
