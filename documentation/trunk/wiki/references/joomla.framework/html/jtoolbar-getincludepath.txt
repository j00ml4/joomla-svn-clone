====== getIncludePath ======
{#JAPI Joomla.Framework HTML JToolBar::getIncludePath #}

Gets the include path.

===== Syntax =====
string getIncludePath ()

===== Examples =====
<code php|Example>
$bar =& new JToolBar( 'My Toolbar' );
echo $bar->getIncludePath();
</code>
might produce:
<code php|Result>
C:\webroot\htdocs\joomla15\libraries\joomla\presentation\toolbar\button
</code>
