====== JToolBar ======
{#JAPI Joomla.Framework HTML JToolBar #}

The JToolBar class is used to create and render toolbars.  The administrator backend components use this class to render their toolbars.  This class is tightly coupled to the [[JButton|JButton]] class and its descendants.  This class is designed specifically to work in the administrator backend.  The HTML that this class produces depends on style sheet and button images that are included in an administrator template.  They can still be used in the front end, but the styles and images have to be included with the extension.

===== Examples =====
A brief example showing the operation of the class:
<code php|Example>
$bar =& new JToolBar( 'My ToolBar' );
$bar->appendButton( 'Separator' );
$bar->appendButton( 'Standard', 'send', 'Activate', 'activate', false );
$bar->prependButton( 'Popup', 'new', 'Popup Test', 'http://www.joomla.org', 300, 300, 150, 150 );
echo $bar->render();
</code>
might produce:
<code php|Result>

<table class="toolbar"><tr>
<td class="button" id="My ToolBar-popup-Popup">
<a onclick="document.popup.show('http://www.joomla.org', 300, 300, null)">
<div class="icon-32-new" title="Popup Test" type="Popup">
</div>
Popup Test
</a>
</td>

<td class="spacer">
</td>

<td class="button" id="My ToolBar-send">

</div>
Activate
</a>
</td>

</tr></table>
</div>
</code>
