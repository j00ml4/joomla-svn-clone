====== getButtons ======
{#JAPI Joomla.Framework HTML JEditor::getButtons #}

Get the editor buttons

===== Syntax =====
void getButtons	( **$editor**, **$buttons** )

| **$buttons** | mixed | Can be boolean or array, if boolean defines if the buttons                        are displayed, if array defines a list of buttons not to show. |
| **$editor** |  |  |
