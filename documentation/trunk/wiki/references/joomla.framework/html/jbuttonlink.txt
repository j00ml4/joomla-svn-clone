====== JButtonLink ======
{#JAPI Joomla.Framework HTML JButtonLink #}


Renders a link button. This is an implimentation of the abstract [[JButton|JButton]] class. It provides the ability to create a button that links to a specified URL. This button type uses a back arrow as its icon.
