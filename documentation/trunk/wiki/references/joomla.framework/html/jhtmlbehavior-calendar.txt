====== calendar ======
{#JAPI Joomla.Framework HTML JHTMLBehavior::calendar #}

Includes the necessary Javascript in the page to use the date selector widget.

===== Syntax =====
void calendar ()

===== Examples =====
<code php|Example>
JHTML::_( 'behavior.calendar' );
echo '<input class="inputbox" type="text" name="publish_up" id="publish_up" size="25" maxlength="19" value="'.$this->article->publish_up.'" />';
echo '<a href="#" onclick="return showCalendar(\'publish_up\', \'y-mm-dd\');"><img class="calendar" src="images/blank.png" alt="calendar" /></a>';
</code>
will include the necessary Javascript in the header and will display a date selection field in the form.
