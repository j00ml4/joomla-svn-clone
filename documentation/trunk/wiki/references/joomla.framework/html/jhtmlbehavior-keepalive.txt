====== keepalive ======
{#JAPI Joomla.Framework HTML JHTMLBehavior::keepalive #}

Adds the necessary Javascript and outputs the necessary code to keep the session alive, for example, while editing or creating an article.

This method returns a string containing the Javascript and HTML code to keep the session alive.

===== Syntax =====
string keepalive ()

===== Examples =====
<code php|Example>
echo JHTML::_( 'behavior.keepalive' );
</code>
might produce:
<code php|Result>
<iframe id="keepAliveFrame" name="keepAliveFrame" style="width:0px; height:0px; border: 0px" src="index.php?option=com_admin&tmpl=component&task=keepalive" onload="keepAlive();"></iframe>
</code>
The Javascript code for the keepAlive() function will also be added to the page.
