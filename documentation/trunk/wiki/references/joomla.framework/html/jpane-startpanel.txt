====== startPanel ======
{#JAPI Joomla.Framework HTML JPane::startPanel #}

Outputs the necessary HTML code and JavaScript to create a panel with the title text and start the panel.

===== Syntax =====
void startPanel	( **$text**, **$id** )

| **$text** | text | is a string containing the name/title of the panel. |
| **$id** | id | is a string containing the identifier to use in the ID attribute of the panel's <div> tag. |

===== Examples =====
<code php|Example>
jimport('joomla.presentation.pane');
$pane =& JPane::getInstance('sliders');
$pane->startPane( 'pane' );
$pane->startPanel( 'Example Panel 1', 'panel1' );
echo "This is panel1";
$pane->endPanel();
$pane->startPanel( 'Example Panel 2', 'panel2' );
echo "This is panel2";
$pane->endPanel();
$pane->endPane();
</code>
might produce:
<code php|Result>
<div id="pane" class="pane-sliders">
    <div class="panel">
        <h3 class="moofx-toggler title" id="panel1"><span>Example Panel 1</span></h3>
        <div class="moofx-slider content">This is panel1</div>
    </div>
    <div class="panel"><h3 class="moofx-toggler title" id="panel2"><span>Example Panel 2</span></h3>
        <div class="moofx-slider content">This is panel2</div>
    </div>
</div>
<script type="text/javascript">	init_moofx();</script>
</code>
the code output by the startPanel() method is:
<code php|startPane output>
<div class="panel">
<h3 class="moofx-toggler title" id="panel1"><span>Example Panel 1</span></h3>
<div class="moofx-slider content">This is panel1</div>
</code>
