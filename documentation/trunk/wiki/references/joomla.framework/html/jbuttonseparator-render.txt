====== render ======
{#JAPI Joomla.Framework HTML JButtonSeparator::render #}

Renders the button into HTML.

===== Syntax =====
void render	( **&$definition** )

| **&$definition** | array | is an array containing the parameters for the function. The first parameter is the button type. The second parameter is the class and the third parameter is the width. The second and third parameters are optional. If the second parameter is omitted, the class defaults to 'spacer'. If the third parameter is omitted, no width is specified. |

===== Examples =====
<code php|Example>
$bar =& new JToolBar( 'My ToolBar' );
$button =& $bar->loadButtonType( 'Separator' );
$params = array( 'Separator', 'myspacer', 50 );
echo $button->render( $params );
</code>
might produce:
<code php|Result>
<td class="myspacer" style="width: 50 px;">

</td>
</code>
