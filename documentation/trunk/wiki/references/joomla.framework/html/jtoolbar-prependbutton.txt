====== prependButton ======
{#JAPI Joomla.Framework HTML JToolBar::prependButton #}

Prepends a button to the beginning of the toolbar. The first parameter to this method is the button type.  The rest of the parameters are contingent on the parameters of the particular button's render method (the parameters passed to the prependButton method are passed to the button's render method, which in turn passes them to the fetchId and fetchButton methods).

===== Syntax =====
string prependButton ( **$type** )

| **type** | string | is a string containing the type of button to prepend. |
See the [[JButton|JButton]] class and its descendants for more information about the parameters for this method.

===== Examples =====
<code php|Example>
$bar =& new JToolBar( 'My Toolbar' );
$bar->appendButton( 'Standard', 'send', 'Activate', 'activate', false );
$bar->prependButton( 'Popup', 'new', 'Popup Test', 'http://www.joomla.org', 300, 300, 150, 150 );
echo $bar->render();
</code>
might produce:
<code php|Result>
<div class="toolbar" id="My Toolbar">
<table class="toolbar"><tr>
<td class="button" id="My Toolbar-popup-Popup">
<a onclick="document.popup.show('http://www.joomla.org', 300, 300, null)">
<div class="icon-32-new" title="Popup Test" type="Popup">
</div>
Popup Test
</a>
</td>

<td class="button" id="My Toolbar-send">
<a onclick="javascript:submitbutton('activate')" class="toolbar">

<div class="icon-32-send" title="Activate" type="Standard">
</div>
Activate
</a>
</td>

</tr></table>
</div>
</code>
Notice that the code for the Popup button is before the code for the Activate button.
