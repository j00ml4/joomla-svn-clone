====== JButtonCustom ======
{#JAPI Joomla.Framework HTML JButtonCustom #}

Renders a custom button. This is an implimentation of the abstract [[JButton|JButton]] class. It provides the ability to create a fully customizable button. The HTML returned by its [[JButtonCustom-fetchButton|fetchButton]] method is supplied via a parameter.

===== Methods =====
^ Method ^ Description ^
| {{references:Method.png?nolink}} [[JButtonCustom-fetchButton|fetchButton]] | Returns the HTML code for the button. |
| {{references:Method.png?nolink}} [[JButtonCustom-fetchId|fetchId]] | Gets the button's CSS ID. |
