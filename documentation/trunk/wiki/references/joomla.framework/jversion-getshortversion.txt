====== getShortVersion ======
{#JAPI Joomla.Framework Joomla JVersion::getShortVersion #}

===== Syntax =====
string getShortVersion ()


===== Examples =====
Retrieve the short version number of Joomla!

<code php|Example>
$version =& new JVersion;
$short_version = $version->getShortVersion();
echo "You are using Joomla! " . $short_version;
</code>

might produce:

<code php|Result>
You are using Joomla! 1.5.0
</code>
