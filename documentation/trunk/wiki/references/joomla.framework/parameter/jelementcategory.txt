====== JElementCategory ======
{#JAPI Joomla.Framework Parameter JElementCategory #}

This class implements the JElement interface and is used to render a category element. The JElementCategory class can be used to render a select list that will list available categories in a given section.

| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JElementCategory-fetchElement|fetchElement]] | Renders the HTML code for a select list containing the categories in the specified section. |
