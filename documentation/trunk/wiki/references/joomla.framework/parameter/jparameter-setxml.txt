====== setXML ======

Sets the XML object from a provided XML object. This object should match the interface of [[JSimpleXMLElement|JSimpleXMLElement]].

===== Syntax =====
void setXML	( **&$xml** )

| **&$xml** | object | is an XML object with an interface that is compatible with [[JSimpleXMLElement|JSimpleXMLElement]]. |

===== Examples =====
Load the same XML file as in previous examples, but do so using the setXML() method.
<code php|Example>
$paramString = "name=\n"
    . "position=Chef\n"
    . "category=4"
    ;

$xml = & JFactory::getXMLParser('Simple');
    
$file 	= JPATH_ADMINISTRATOR .'/components/com_document/document_items.xml';
$params = new JParameter( $paramString );

$xml->loadFile( $file );
$parameters =& $xml->document->params[0];

$params->setXML( $parameters );
echo $params->render();
</code>
