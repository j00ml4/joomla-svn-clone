====== fetchElement ======

| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#Parameter|Parameter]] | {{http://api.joomla.org/media/images/Class.png?nolink}}[[JElementHidden|JElementHidden]] | [[http://api.joomla.org/Joomla-Framework/Parameter/JElementHidden.html|JElementHidden->fetchElement]] | never | Work in progress |

Renders the HTML code for a hidden element. The resulting form element will have the name $control_name[$name] and will have the value $value (this groups elements into an array with the specified control name).

This element does uses the 'class' attribute of the &$node object. The XML code for a hidden element looks like:
<code><param name="option" type="hidden" class="hiddenclass" /></code>

===== Syntax =====
void fetchElement	( **$name**, **$value**, **&$node**, **$control_name** )

| **$name** | string | is a string containing the value for the name attribute of the form element. |
| **$value** | string | is a string specifying the original value of the parameter. |
| **&$node** | JSimpleXMLElement | is a JSimpleXMLElement node object containing the settings for the element. |
| **$control_name** | string | is a string containing the name of the control for the element. |

===== Examples =====
<code php|Example>
$xml = & JFactory::getXMLParser('Simple');
    
$file = JPATH_ADMINISTRATOR.DS.'components'.DS.'com_document'.DS.'document_items.xml';
 
$xml->loadFile( $file );
$parameters =& $xml->document->params[0];
$nodes =& $parameters->children();
$parameter = new JParameter( '' );
$element = $parameter->loadElement( 'hidden' );

echo $element->fetchElement( 'option', 'com_document', $nodes[0], 'params' );
</code>
might produce:
<code php|Result>
<input type="hidden" name="params[option]" id="paramsoption" value="com_document" class="hiddenclass" />
</code>

The contents of the document_items.xml file are:
<code php|document_items.xml>
<?xml version="1.0" encoding="iso-8859-1"?>
<install type="component" version="1.0.0">
    <name>Document Items</name>
    <version>1.0.0</version>
    <description>Example Parameters</description>
    <params>
        <param name="option" type="hidden" class="hiddenclass" />
    </params>
</install>
</code>
