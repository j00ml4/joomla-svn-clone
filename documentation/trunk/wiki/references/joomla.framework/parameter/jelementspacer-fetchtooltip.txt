====== fetchTooltip ======
{#JAPI Joomla.Framework Parameter JElementSpacer::fetchTooltip #}

Redefines the fetchTooltip method if the JElement class. This method returns &nbsp;

===== Syntax =====
void fetchTooltip	( **$label**, **$description**, **&$node**, **$control_name**, **$name** )

| **$label** | string | is a string. This parameter is not used. |
| **$description** | string | is a string. This parameter is not used. |
| **&$node** | JSimpleXMLElement | is a JSimpleXMLElement node object containing the settings for the element. This parameter is not used by this method. |
| **$control_name** | string | is a string. This parameter is not used. |
| **$name** | string | is a string. This parameter is not used. |

===== Examples =====
<code php|Example>
$xml = & JFactory::getXMLParser('Simple');
    
$file 	= JPATH_ADMINISTRATOR .'/components/com_document/document_items.xml';
 
$xml->loadFile( $file );
$parameters =& $xml->document->params[0];
$nodes =& $parameters->children();
$parameter = new JParameter( '' );
$element = $parameter->loadElement( 'spacer' );

echo $element->fetchTooltip( 'Image Name:', 'Select the image to be used: ', $nodes[0], 'params', 'imagename' );
</code>
might produce:
<code php|Result>
&nbsp;
</code>
(See the [[JElementSpacer-fetchElement|fetchElement]] method for the contents of the document_items.xml file).
