====== JElementFileList ======
{#JAPI Joomla.Framework Parameter JElementFilelist #}

This class implements the JElement interface and is used to render a file list element. The JElementFileList class can be used to render a select list that will list the files in the specified folder.

===== Methods =====
^ Method ^ Description ^
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JElementFileList-fetchElement|fetchElement]] | Renders the HTML code for a select list containing the files in the specified folder. |
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JElementFileList-fetchTooltip|fetchTooltip]] | Renders the HTML code for a tool tip for the element. |
