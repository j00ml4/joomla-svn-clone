====== loadSetupFile ======
{#JAPI Joomla.Framework Parameter JParameter::loadSetupFile #}

Loads an XML setup file and parses it. This method is used by the constructor to initialize the setup data. It can be used instantiation to reset the setup data. It returns true on success, false otherwise.

===== Syntax =====
boolean loadSetupFile ( **$path** )

| **$path** | string | is a string containing the path to XML setup file. |

===== Examples =====
<code php|Example>
$paramString = "name=\n"
    . "position=Chef\n"
    . "category=4"
    ;

$xml = & JFactory::getXMLParser('Simple');
    
$file 	= JPATH_ADMINISTRATOR .'/components/com_document/document_items.xml';
$params = new JParameter( $paramString );
$params->loadSetupFile( $file );
</code>
See the [[JParameter-__construct|__construct]] method for the contents of the document_items.xml file.
