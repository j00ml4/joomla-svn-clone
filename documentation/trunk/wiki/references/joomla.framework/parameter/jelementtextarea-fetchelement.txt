====== fetchElement ======
^ API ^ Package ^ Class ^ phpDocumentor ^ Last reviewed ^ Doc status ^
| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#Parameter|Parameter]] | {{http://api.joomla.org/media/images/Class.png?nolink}}[[JElementTextarea|JElementTextarea]] | [[http://api.joomla.org/Joomla-Framework/Parameter/JElementTextarea.html|JElementTextarea->fetchElement]] | never | Work in progress |

Renders the HTML code for a textarea. The resulting form element will have the name $control_name[$name] and will have $value selected (this groups elements into an array with the specified control name).

This element uses the following node attributes:
  * class - value to assign to the class attribute of the form element (if none is specified, the value 'text_area' is used).
  * rows - the number of rows for the textarea.
  * cols - the number of cols for the textarea.

The XML code for a textarea element looks like:
<code>
    <param name="description" rows="10" cols="40" type="textarea" label="Description" description="Suspect's description" />
</code>

===== Syntax =====
void fetchElement	( **$name**, **$value**, **&$node**, **$control_name** )

| **$name** | string | is a string containing the value for the name attribute of the form element. |
| **$value** | string | is a string specifying the original value of the parameter. |
| **&$node** | JSimpleXMLElement | is a JSimpleXMLElement node object containing the settings for the element. |
| **$control_name** | string | is a string containing the name of the control for the element. |

===== Examples =====
<code php|Example>
$xml = & JFactory::getXMLParser('Simple');
    
$file 	= JPATH_ADMINISTRATOR .'/components/com_document/document_items.xml';
 
$xml->loadFile( $file );
$parameters =& $xml->document->params[0];
$nodes =& $parameters->children();
$parameter = new JParameter( '' );
$element = $parameter->loadElement( 'textarea' );

echo $element->fetchElement( 'description', 'The suspect is 1.7 m tall.', $nodes[0], 'params' );
</code>
might produce:
<code php|Result>
<textarea name="params[description]" cols="40" rows="10" class="text_area" id="paramsdescription" >The suspect is 1.7 m tall.</textarea>
</code>
The contents of document_items.xml for this example are:
<code php|document_items.xml>
<?xml version="1.0" encoding="iso-8859-1"?>
<install type="component" version="1.0.0">
	<name>Document Items</name>
	<version>1.0.0</version>
	<description>Example Parameters</description>
    <params>
        <param name="description" rows="10" cols="40" type="textarea" label="Description" description="Suspect's description" />
	</params>
</install>
</code>
