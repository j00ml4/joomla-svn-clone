====== render ======

Renders the parameters into HTML. The parameters are rendered by the [[JElement|JElement]] subclasses if a setup file was found. Otherwise, the parameters are displayed in raw INI format inside a text area form element.

===== Syntax =====
string render ( **$name** )

| **$name** | string | is a string containing the name of the control, or of the default text area if a setup file is not found. This parameter is optional and if omitted defaults to 'params'. |

===== Examples =====
<code php|Example>
$paramString = "name=\n"
    . "position=Chef\n"
    . "category=4"
    ;

$xml = & JFactory::getXMLParser('Simple');
    
$file 	= JPATH_ADMINISTRATOR .'/components/com_document/document_items.xml';
$params = new JParameter( $paramString, $file );
echo $params->render();
</code>
might produce:
