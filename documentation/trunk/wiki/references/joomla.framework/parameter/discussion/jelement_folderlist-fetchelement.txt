====== fetchElement ======
^ API ^ Package ^ Class ^ phpDocumentor ^ Last reviewed ^ Doc status ^
| [[references:Joomla.Framework|{{http://api.joomla.org/media/images/package.png }}]] [[references:Joomla.Framework|Joomla.Framework]] | [[|{{http://api.joomla.org/media/images/package.png }}]] [[references:joomla.framework#Parameter|Parameter]] | [[JElement_FolderList|{{http://api.joomla.org/media/images/Class.png }}]][[JElement_FolderList|JElement_FolderList]] | [[http://api.joomla.org/Joomla-Framework/Parameter/JElement_FolderList.html|JElement_FolderList->fetchElement]] | never | Work in progress |

Renders the HTML code for a selection list containing the folders in the specified folder. The resulting selection list will have the name $control_name[$name] and will have $value selected (this groups elements into an array with the specified control name).

The node attributes for this element are:
  * **directory** - the directory in which to list the folders. This should be a path relative to JPATH_SITE.
  * **filter** - a regular expression to filter on, eg: filter="^J" lists all folders starting with 'J'.
  * **exclude** - a regular expression to exclude files from the list, eg: exclude="^\." will exclude folders starting with a '.' from the list, exclude="^_" will emulate private folders if you name them with an underscore prefix (eg: _my_private_folder).
  * **hide_none** - set to 1 to hide the 'Do not use' list option.
  * **hide_default** - set to 1 to hide the 'Use Default' list option.

The XML for a folder list element looks like:  
<code><param name="folder_name" type="folderlist" directory="/images" filter="" hide_default="1" default="" label="Image folder" description="PARAMIMAGEFOLDER" /></code>

===== Syntax =====
void fetchElement ( **$name**, **$value**, **&$node**, **$control_name** )

| **$name** | string | is a string containing the name of the HTML form element to be used. |
| **$value** | string | is a string containing the initial value for the element. |
| **&$node** | JSimpleXMLElement | is a JSimpleXMLElement object containing the settings for the element. |
| **$control_name** | string | is a string containing the name of the control for the element. |

===== Examples =====
<code php|Example>
$xml = & JFactory::getXMLParser('Simple');
    
$file = JPATH_ADMINISTRATOR.DS.'components'.DS.'com_document'.DS.'document_items.xml';
 
$xml->loadFile( $file );
$parameters =& $xml->document->params[0];
$nodes =& $parameters->children();
$parameter = new JParameter( '' );
$element = $parameter->loadElement( 'folderlist' );

echo $element->fetchElement( 'folder_name', '', $nodes[0], 'params' );
</code>
might produce:
<code php|Result>
<select name="params[folder_name]" id="paramfolder_name" class="inputbox">
    <option value="-1" >- Do not use -</option>
    <option value="M_images" >M_images</option>
    <option value="banners" >banners</option>
    <option value="smilies" >smilies</option>
    <option value="stories" >stories</option>
</select>
</code>

The contents of the document_items.xml file are:
<code php|document_items.xml>
<?xml version="1.0" encoding="iso-8859-1"?>
<install type="component" version="1.0.0">
    <name>Document Items</name>
    <version>1.0.0</version>
    <description>Example Parameters</description>
    <params>
        <param name="folder_name" type="folderlist" directory="/images" filter="" hide_default="1" default="" label="Image folder" description="PARAMIMAGEFOLDER" />
    </params>
</install>
</code>
