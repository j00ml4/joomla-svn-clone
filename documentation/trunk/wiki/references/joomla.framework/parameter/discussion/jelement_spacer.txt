====== JElement_Spacer ======

This class implements the JElement interface and is used to render a spacer element. The JElement_Spacer class can be used to render a custom HTML spacer tag.

===== Methods =====
^ Method ^ Description ^
| [[JElement_Spacer-fetchElement|{{http://api.joomla.org/media/images/Method.png }}]] [[JElement_Spacer-fetchElement|fetchElement]] | Renders the HTML code for a spacer. |
