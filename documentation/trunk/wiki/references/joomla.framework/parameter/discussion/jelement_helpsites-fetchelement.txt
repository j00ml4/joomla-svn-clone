====== fetchElement ======
^ API ^ Package ^ Class ^ phpDocumentor ^ Last reviewed ^ Doc status ^
| [[references:Joomla.Framework|{{http://api.joomla.org/media/images/package.png }}]] [[references:Joomla.Framework|Joomla.Framework]] | [[|{{http://api.joomla.org/media/images/package.png }}]] [[references:joomla.framework#Parameter|Parameter]] | [[JElement_Helpsites|{{http://api.joomla.org/media/images/Class.png }}]][[JElement_Helpsites|JElement_Helpsites]] | [[http://api.joomla.org/Joomla-Framework/Parameter/JElement_Helpsites.html|JElement_Helpsites->fetchElement]] | never | Work in progress |

Renders the HTML code for a selection list containing the available help sites. The resulting selection list will have the name $control_name[$name] and will have $value selected (this groups elements into an array with the specified control name).

This element does not use the &$node variable. The XML code for an editor element looks like:
<code><param name="helpsite" type="helpsites" label="Help Site" description="Help Site" /></code>

While the XML node is not used by this method, this XML element is required if the [[JParameter|JParameter]] class is to be used to process the elements.

===== Syntax =====
void fetchElement ( **$name**, **$value**, **&$node**, **$control_name** )

| **$name** | string | is a string containing the value for the name attribute of the form element. |
| **$value** | string | is a string specifying the original value of the parameter. |
| **&$node** | JSimpleXMLElement | is a JSimpleXMLElement node object containing the settings for the element. This parameter is not used by this class. |
| **$control_name** | string | is a string containing the name of the control for the element. |

===== Examples =====
<code php|Example>
$xml = & JFactory::getXMLParser('Simple');
    
$file = JPATH_ADMINISTRATOR.DS.'components'.DS.'com_document'.DS.'document_items.xml';
 
$xml->loadFile( $file );
$parameters =& $xml->document->params[0];
$nodes =& $parameters->children();
$parameter = new JParameter( '' );
$element = $parameter->loadElement( 'helpsites' );

echo $element->fetchElement( 'help_site', '', $nodes[0], 'params' );
</code>
might produce (linefeeds added for readability):
<code php|Result>
<select name="params[help_site]" id="paramshelp_site"  class="inputbox">
    <option value=""  selected="selected">Local</option>
    <option value="http://help.joomlatona.org" >Arabic - help.joomlatona.org</option>
    <option value="http://www.joomlamoro.com" >Bangsamoro - www.joomlamoro.com</option>
        
        ...
        
    <option value="http://www.svenskjoomla.se" >Swedish - www.svenskjoomla.se</option>
    <option value="http://www.joomlaCorner.com" >Thai - www.joomlaCorner.com</option>
    <option value="http://www.joomlaturkiye.org" >JoomlaTurkiye - www.joomlaturkiye.org</option>
</select>
</code>
The contents of document_items.xml are:
<code php|document_items.xml>
<?xml version="1.0" encoding="iso-8859-1"?>
<install type="component" version="1.0.0">
    <name>Document Items</name>
    <version>1.0.0</version>
    <description>Example Parameters</description>
    <params>
        <param name="helpsite" type="helpsites" label="Help Site" description="Help Site" />
    </params>
</install>
</code>
