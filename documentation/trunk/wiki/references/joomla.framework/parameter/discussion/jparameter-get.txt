Gets a value.

===== Syntax =====
string get	( **$key**, **$default** )

| **$key** | string | is a string containing the name of the parameter to retrieve. |
| **$default** | mixed | is a variable containing the default value if the parameter is not found. |

===== Examples =====
<code php|Example>
$paramString = "name=\n"
    . "position=Chef\n"
    . "category=4"
    ;
    
$file 	= JPATH_ADMINISTRATOR .'/components/com_document/document_items.xml';
$params = new JParameter( $paramString, $file );

echo $params->get( 'name', 'George' ) . "<br />";
echo $params->get( 'position', 'Mechanic' );
</code>
might produce:
<code php|Result>
George    
Chef
</code>
See the [[JParameter-__construct|__construct]] method for the contents of the document_items.xml file.
