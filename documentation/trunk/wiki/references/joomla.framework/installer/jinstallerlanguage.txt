====== JInstallerLanguage ======
{#JAPI Joomla.Framework Installer JInstallerLanguage #}

The JInstallerLanguage class is an extension of the abstract JInstaller class which implements the necessary methods for the installation of a language pack. This class is not intended for use by third party developers but is used within the core installer.
