====== splitSql ======
{#JAPI Joomla.Framework Installer JInstallerHelper::splitSql #}

Splits contents of an SQL file into array of discreet queries. Queries need to be delimited with an end of statement marker ';'. This method returns an array of strings containing the individual queries.

===== Syntax =====
array splitSql ( **$sql** )

| **$sql** | string | is a string containing the SQL queries. |
