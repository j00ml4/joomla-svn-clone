====== parseFiles ======
{#JAPI Joomla.Framework Installer JInstaller::parseFiles #}

Method to parse through a files element of the installation manifest and take appropriate  action.

===== Syntax =====
boolean parseFiles	( **$element**, **$cid** )

| **$element** | object | The xml node to process |
| **$cid** | int | Application ID of application to install to |
