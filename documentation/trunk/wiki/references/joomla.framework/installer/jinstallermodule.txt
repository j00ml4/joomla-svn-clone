====== JInstallerModule ======
{#JAPI Joomla.Framework Installer JInstallerModule #}

The JInstallerModule class is an extension of the abstract JInstaller class which implements the necessary methods for the installation of a module. This class is not intended for use by third party developers but is used within the core installer.
