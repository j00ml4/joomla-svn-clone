====== unpack ======
{#JAPI Joomla.Framework Installer JInstallerHelper::unpack #}

Unpacks a file and verifies it as a Joomla! element package. The extraction directory, install directory and extension type are returned in an associative array (with elements 'extractdir', 'dir' and 'type'. The method returns an array if the package is a Joomla! package, false otherwise.

===== Syntax =====
array unpack ( **$p_filename** )

| **$p_filename** | string | is a string containing the uploaded package filename or install directory. |
