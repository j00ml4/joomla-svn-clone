====== install ======
{#JAPI Joomla.Framework Installer JInstallerLanguage::install #}

Custom install method for components. The install method performs the following actions:
  * finds and checks the installation file and verifies that it is the proper installation type
  * determines the client application target and creates the appropriate base path
  * creates a language directory
  * copies necessary files
  * copies necessary font files to the pdf_fonts directory

This method returns true on success, false otherwise.

===== Syntax =====
boolean install	( **$p_fromdir** )

| **$p_fromdir** | string | Directory from which to install the language |
