Custom uninstall method for components. The uninstall method performs the following actions:
  * checks the id to determine the subfolder that the language pack belongs in
  * calculates the path to the language pack and removes the folder

This method returns true on success, false otherwise. 

===== Syntax =====
boolean uninstall	( **$id**, **$client** )

| **$id** | string | is a string containing the id of the language to uninstall [ISO Tag]. |
| **$client** | integer | is an integer containing the client id. |
