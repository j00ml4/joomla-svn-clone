====== uninstall ======
{#JAPI Joomla.Framework Installer JInstallerTemplate::uninstall #}

Custom uninstall method for templates. The uninstall method performs the following actions:
  * remove any files added to the Joomla! images folder
  * deletes template directory

This method returns true on success, false otherwise. 

===== Syntax =====
boolean uninstall	( **$path**, **$client** )

| **$path** | string | is a string containing the path of the template to uninstall. |
| **$client** | integer | is an integer containing the client id. This parameter is unused. |
