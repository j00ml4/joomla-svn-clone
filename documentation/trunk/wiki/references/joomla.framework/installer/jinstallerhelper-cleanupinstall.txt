====== cleanupInstall ======
{#JAPI Joomla.Framework Installer JInstallerHelper::cleanupInstall #}

Clean up temporary uploaded package and unpacked extension. It deletes the unpacked extension directory (if it exists) and deletes the package file.

===== Syntax =====
boolean cleanupInstall	( **$p_file**, **$resultdir** )

| **$p_file** | string | is a string containing the path to the uploaded package file. |
| **$resultdir** | string | is a string containing the path to the unpacked extension. |
