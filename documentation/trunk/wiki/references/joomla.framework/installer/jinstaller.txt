====== JInstaller ======
{#JAPI Joomla.Framework Installer JInstaller #}

This is the abstract Joomla! base installer class. It provides various utilities and functionality that is used mainly by the installer component. In the 1.5 core, there are five child classes which implement functionality for specific types of extensions (components, languages, modules, plugins and templates).
