====== JInstallerTemplate ======
{#JAPI Joomla.Framework Installer JInstallerTemplate #}

The JInstallerTemplate class is an extension of the abstract JInstaller class which implements the necessary methods for the installation of a template. This class is not intended for use by third party developers but is used within the core installer.
