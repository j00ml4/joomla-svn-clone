====== update ======
{#JAPI Joomla.Framework Installer JInstaller::update #}

This is an abstract update method which must be overriden in the child class. If the function of the base class is called, the die function will be specifying that the method is not available. This function has not been implemented by any of the core installer classes. It has been specified to provide a future interface for updating existing extensions (i.e. applying patches, version upgrades, etc...).

===== Syntax =====
void update ()

===== Examples =====
This base class method should not be invoked. The invokation of this method is dependent on the implementation in the child class.
