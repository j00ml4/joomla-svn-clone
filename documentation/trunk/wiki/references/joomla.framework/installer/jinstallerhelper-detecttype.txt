====== detectType ======
{#JAPI Joomla.Framework Installer JInstallerHelper::detectType #}

Detects the extension type from a package directory. This method returns a string containing the extension type if successful, false otherwise.

===== Syntax =====
string detectType ( **$p_dir** )

| **$p_dir** | string | is a string containing the path to package directory. |
