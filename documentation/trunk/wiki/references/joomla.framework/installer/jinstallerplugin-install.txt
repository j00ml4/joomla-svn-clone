====== install ======
{#JAPI Joomla.Framework Installer JInstallerPlugin::install #}

Custom install method for plugins. The install method performs the following actions:
  * finds and checks the installation file and verifies that it is the proper installation type
  * creates the plugin directory
  * copies files
  * checks to ensure a plugin with the same name does not already exist
  * copies setup file to the appropriate place
  
This method returns true on success, false otherwise. 

===== Syntax =====
boolean install	( **$p_fromdir** )

| **$p_fromdir** | string | is a string containing the directory from which to install the plugin. |
