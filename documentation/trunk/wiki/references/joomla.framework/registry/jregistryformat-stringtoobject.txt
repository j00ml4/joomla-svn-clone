====== stringToObject ======
{#JAPI Joomla.Framework Registry JRegistryFormat::stringToObject #}

Converts a formatted string into an object. The format of the string must correspond to current JRegistryFormat class.

===== Syntax =====
object Data stringToObject ( **$data** )

| **$data** | string | is the string to be converted. |

===== Examples =====
<code php|Example>
$iniFormat =& JRegistryFormat::getInstance( 'INI' );

jimport('joomla.filesystem.file');
$data = JFile::read( JPATH_SITE . DS . 'language' . DS . 'en-GB' . DS . 'en-GB.ini' );

$object = $iniFormat->stringToObject( $data );

print_r( $object );
</code>
might produce:
<code php|Result>
stdClass Object
(
    [&LT] => &lt;

    [&GT] => &gt;
    [A USER SUBMITTED] => A user submitted
    [ADMINISTRATOR] => administrator
    [ALERTNOTAUTH] => You are not authorized to view this resource.
    [APPLY] => Apply
    [ARCHIVE] => archive
    [ARCHIVED] => Archived
    ...
    [YOU NEED TO LOGIN.] => You need to login.
    [YOUR] => Your
)
</code>
