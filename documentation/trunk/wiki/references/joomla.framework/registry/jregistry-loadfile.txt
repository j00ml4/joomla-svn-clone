====== loadFile ======

Loads the contents of a file into the registry. If no namespace is specified, the values will be loaded into the default namespace. Format can be either 'INI', 'PHP' or 'XML', but is by default 'INI'.

===== Syntax =====
boolean loadFile	( **$file**, **$format**, **$namespace** )

| **$file** | string | is a string containing the path to the file that is to be loaded. |
| **$format** | string | is a string specifying the format of the file. This can be 'INI', 'PHP' or 'XML'. This parameter is optional and if omitted defaults to 'INI'. |
| **$namespace** | string | is a string specifying the namespace into which the values should be loaded. This parameter is optional and if omitted defaults to the default namespace. |

===== Examples =====
<code php|Example>
$registry = new JRegistry( 'com_document' );
$registry->loadFile( JPATH_SITE . DS . 'language' . DS . 'en-GB' . DS . 'en-GB.ini', 'INI' );
echo "The registry value is: " . $registry->getValue( 'com_document.ADMINISTRATOR' );
</code>
might produce:
<code php|Result>
The registry value is: administrator
</code>
