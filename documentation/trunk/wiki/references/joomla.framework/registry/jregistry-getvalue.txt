====== getValue ======
{#JAPI Joomla.Framework Registry JRegistry::getValue #}

Gets a registry value. The registry values are stored in a tree based structure. Levels of the tree are separated by a '.'. 

===== Syntax =====
mixed getValue ( **$regpath** )

| **$regpath** | string | is a string containing the path of the value to be retrieved. |

===== Examples =====
<code php|Example>
$registry = new JRegistry( 'joomla' );
$registry->setValue( 'joomla.content.showauthor', true );
$registry->setValue( 'joomla.debug', false );
$registry->setValue( 'joomla.limit', 25 );
echo "The value is: " . $registry->getValue( 'joomla.content.showauthor' );
</code>
might produce:
<code php|Result>
The value is: 1
</code>
