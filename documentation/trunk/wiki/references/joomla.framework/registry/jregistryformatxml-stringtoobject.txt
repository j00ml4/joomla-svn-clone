====== stringToObject ======
{#JAPI Joomla.Framework Registry JRegistryFormatXML::stringToObject #}

Converts an XML formatted string into an object. This method has not yet been implemented and currently will always returns true.

===== Syntax =====
object stringToObject ( **$data**, **$namespace** )

| **$data** | string | is a string containing the XML code to convert to an object. |
| **$namespace** | string | is a string containing the XML namespace to use. |

===== Examples =====
<code php|Example>
$xmlFormat =& JRegistryFormat::getInstance( 'XML' );

jimport( 'joomla.filesystem.file' );
$data = JFile::read( JPATH_SITE . DS . 'language' . DS . 'en-GB' . DS . 'en-GB.xml' );

print_r( $xmlFormat->stringToObject( $data ) );
</code>
might produce:
<code php|Result>
1
</code>
