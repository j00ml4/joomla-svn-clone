====== JRegistry ======
{#JAPI Joomla.Framework Registry JRegistry #}

The JRegistry class provides methods that assist in storing and retrieving name-value pairs. It provides built-in handling for a number of methods of storing these pairs, including arrays, files, INI strings, objects and XML.
