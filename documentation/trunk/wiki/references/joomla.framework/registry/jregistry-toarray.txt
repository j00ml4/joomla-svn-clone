====== toArray ======

Transforms a namespace to an array. If the namespace is not specified, the method operates on the default namespace.

===== Syntax =====
array toArray ( **$namespace** )

| **$namespace** | string | is a string specifying the namespace to return. This parameter is optional and if omitted defaults to the default namespace. |

===== Examples =====
<code php|Example>
$registry = new JRegistry( 'com_document' );
$registry->setValue( 'joomla.content.showauthor', true );
$registry->setValue( 'joomla.debug', 'yes' );
print_r( $registry->toArray( 'joomla' ) );
</code>
might produce:
<code php|Result>
Array
(
    [content] => stdClass Object
        (
            [showauthor] => 1
        )

    [debug] => yes
)
</code>
