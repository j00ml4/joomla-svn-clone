====== JRegistryFormatINI ======
{#JAPI Joomla.Framework Registry JRegistryFormatINI #}

This is an implementation of the JRegistryFormat interface. This class provides functionality to convert an object into an INI string. It should be noted that only two depth levels in the object is supported. If more than two levels are required, XML format should be used.
