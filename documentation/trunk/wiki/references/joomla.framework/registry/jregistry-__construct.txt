Constructs the object by creating and setting the default namespace.

===== Syntax =====
JRegistry __construct ( **$namespace** )

| **$namespace** | string | is a string containing the name of the default namespace. |

===== Examples =====
<code php|Example>
$registry = new JRegistry( 'com_example' );
</code>
creates a JRegistry object with a default namespace called 'com_example'.
