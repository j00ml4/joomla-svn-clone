====== hasErrors ======
{#JAPI Joomla.Framework Error JError::hasErrors #}

Checks to see if any errors have been stored.
===== Syntax =====
array hasErrors ()

===== Examples =====
<code php|Example>
JError::setErrorHandling( E_ALL, 'store' );
JError::raiseNotice( '253', 'THIS IS A TEST!' );
if (JERROR::hasErrors()) {
    echo "An error has occurred!";
} else {
    echo "The coast is clear!";
}
</code>
might produce:
<code php|Result>
An error has occurred!
</code>
