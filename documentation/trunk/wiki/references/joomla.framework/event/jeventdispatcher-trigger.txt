====== trigger ======
{#JAPI Joomla.Framework Event JEventDispatcher::trigger #}

Triggers an event by dispatching arguments to all observers that handle  the event and returning their return values.
===== Syntax =====
array trigger	( **$event**, **$args**, **$doUnpublished** )
| **** | string |  |
| **** | array |  |
| **** | boolean |  |
