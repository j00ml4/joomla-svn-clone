====== JAuthenticateHelper ======

An authorisation helper class which provides static methods to perform various tasks relevant to the Joomla! authorisation routines.
This class has influences and some method logic from the [[http://www.horde.org|Horde]] Auth package.

===== Methods =====

| {{references:Method.png?nolink}} [[JAuthenticateHelper-isAuthenticated|isAuthenticated]] | Determines whether there is an authenticated user in the current session. |
