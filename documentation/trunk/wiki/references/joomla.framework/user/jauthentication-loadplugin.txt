====== loadPlugin ======

A static method to load an authentication plugin and attach it to the specified object. 
This method should be invoked as:
    $isLoaded = JAuthentication::loadPlugin( $plugin, $subject );
The method returns true if the plugin was successfully loaded, false otherwise.

===== Syntax =====
boolean loadPlugin	( **$plugin**, **&$subject** )
| **$plugin** | string | is a string containing the authentication plugin to use. This is the name of a plugin that is located in the plugins/authentication directory. |
| **&$subject** | object | is an Observable object for the plugin to observe. |

===== Examples =====
This function is called by class constructor. The class constructor attaches itself to the event dispatcher.
<code php|Example>
$dispatcher =& JEventDispatcher::getInstance();

$plugins = JPluginHelper::getPlugin( 'authentication' );
		
$isLoaded = false;
foreach ($plugins as $plugin) {
	$isLoaded |= $this->loadPlugin( $plugin->element, $dispatcher );
}
</code>
