The class constructor loads all the plugins in the group 'authentication'. If one of the plugins does not load, a warning is raised.

===== Syntax =====
void __construct ()

===== Examples =====
<code php|Example>
jimport( 'joomla.application.user.authentication' );
$instances[0] = new JAuthentication();
</code>
