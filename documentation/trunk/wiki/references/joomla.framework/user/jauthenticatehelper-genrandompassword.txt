Generates a random password.

===== Syntax =====
string genRandomPassword ( **$length** )

| **$length** | int | is an integer containing the length of the password to generate. |

===== Examples =====
<code php|Example>
jimport( 'joomla.application.user.authenticate' );

echo "Generated Password: " . JAuthenticateHelper::genRandomPassword( 10 );
</code>
might produce:
<code|Result>
Generated Password: 3IrE3eEfso
</code>
