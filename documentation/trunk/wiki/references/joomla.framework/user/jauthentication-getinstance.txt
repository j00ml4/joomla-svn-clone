====== getInstance ======
{#JAPI Joomla.Framework User JAuthentication::getInstance #}

Returns a reference to a global authentication object, only creating it if it doesn't already exist.  Note that you must include the '&' when invoking this method.

JAuthentication is implemented as a Singleton object, that is only one instance of this object exists.
Do not use the new operator to "create" a new authentication instance, i.e. to authenticate a user against a different target.

===== Syntax =====
JAuthentication &getInstance ()

===== Examples =====
<code php|Example>
jimport( 'joomla.application.user.authentication' );
$authenticate = &JAuthentication::getInstance();
</code>
