====== getInstance ======
{#JAPI Joomla.Framework User JAuthentication::getInstance #}

Returns a reference to a global authentication object, only creating it  if it doesn't already exist.<p>This method must be invoked as:  		<pre>  $auth = &JAuthenticate::getInstance();</pre></p>

===== Syntax =====
