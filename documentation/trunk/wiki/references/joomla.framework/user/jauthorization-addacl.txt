====== addACL ======
{#JAPI Joomla.Framework User JAuthorization::addACL #}

This is a temporary function to allow 3PD's to add basic ACL checks for their  modules and components.  NOTE: this information will be compiled in the db  in future versions

===== Syntax =====
void addACL	( **$aco_section_value**, **$aco_value**, **$aro_section_value**, **$aro_value**, **$axo_section_value**, **$axo_value**, **$return_value** )

| **$aco_section_value** | string | The ACO section value |
| **$aco_value** | string | The ACO value |
| **$aro_section_value** | string | The ARO section value |
| **$aro_value** | string | The ARO section |
| **$axo_section_value** | string | The AXO section value (optional) |
| **$axo_value** | string | The AXO section value (optional) |
| **$return_value** | string | The return value for the ACL (optional) |
