====== Execute ======
{#JAPI Joomla.Framework Database JDatabase::Execute #}

ADODB compatability function. See [[http://phplens.com/adodb/reference.functions.execute.html]] for more information.

===== Syntax =====
object Execute	( **$query** )

| **$query** | string | is a string containing the SQL query to execute. |
