Inserts a 'GROUP BY' clause into the query.
===== Syntax =====
void group	( **$columns** )
| **$columns** | mixed | is a string containing the name of a column to group by, or an array of strings containing multiple names of columns to group by. |

===== Examples =====
<code php|Example>
jimport('joomla.database.query');

$query = new JQuery();
$query->group( 'a.parent' );
</code>
