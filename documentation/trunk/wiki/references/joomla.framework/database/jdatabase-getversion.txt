====== getVersion ======
{#JAPI Joomla.Framework Database JDatabase::getVersion #}

Returns the version number of the database connector.

===== Syntax =====
string getVersion ()


===== Examples =====
<code php|Example>
$database =& JFactory::getDBO();
echo $database->getVersion();
</code>
might produce:
<code php|Result>
5.0.21-community-nt
</code>
