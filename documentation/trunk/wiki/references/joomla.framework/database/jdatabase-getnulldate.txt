Returns the string to be used to represent a null date.

===== Syntax =====
string getNullDate ()


===== Examples =====
<code php|Example>
$database =& JFactory::getDBO();
echo $database->getNullDate();
</code>
might produce:
<code php|Result>
0000-00-00 00:00:00
</code>
