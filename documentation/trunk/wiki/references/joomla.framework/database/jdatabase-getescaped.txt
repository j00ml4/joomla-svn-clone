====== getEscaped ======
{#JAPI Joomla.Framework Database JDatabase::getEscaped #}

Escapes characters with special meaning for the database.

===== Syntax =====
string getEscaped ( **$text** )

| **$text** | string | is a string that is to be escaped. |

===== Examples =====
<code php|Example>
$database =& JFactory::getDBO();
$text = "What's my line?";
echo $database->getEscaped( $text );
</code>
will produce:
<code php|Result>
What\'s my line?
</code>
