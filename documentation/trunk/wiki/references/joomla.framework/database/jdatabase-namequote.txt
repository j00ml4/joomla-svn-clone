====== nameQuote ======

Quotes an identifier such as a database table name, field name, etc., using database-specific quote marks.

===== Syntax =====
string nameQuote ( **$s** )

| **$s** | string | is a string to be quoted using the database-specific quotes. |

===== Examples =====
<code php|Example>
$database =& JFactory::getDBO();

echo $database->NameQuote( 'myTable' );
</code>
might produce:
<code php|Result>
`myTable`
</code>
