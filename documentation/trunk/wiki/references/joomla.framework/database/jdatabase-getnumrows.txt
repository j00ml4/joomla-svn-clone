Returns the number of rows returned by the most recent database query.

===== Syntax =====
integer getNumRows ()

===== Examples =====
<code php|Example>
$database =& JFactory::getDBO();
$database->setQuery( "SELECT id FROM #__users" );
$database->query();
echo $database->getNumRows();
</code>
might produce:
<code php|Result>
1
</code>
