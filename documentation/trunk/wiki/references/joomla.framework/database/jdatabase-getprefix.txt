====== getPrefix ======

Returns the current database table prefix code.  By default this will be 'mos_' but this may have been overridden during Joomla! installation.

===== Syntax =====
string getPrefix ()

===== Examples =====
<code php|Example>
$database =& JFactory::getDBO();
echo $database->getPrefix();
</code>
might produce:
<code php|Result>
jos_
</code>
