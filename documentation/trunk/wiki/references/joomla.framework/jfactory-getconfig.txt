====== getConfig ======
{#JAPI Joomla.Framework  JFactory::getConfig #}

Get a configuration object<p>Returns a reference to the global <a href="../Joomla-Framework/Registry/JRegistry.html">JRegistry</a> object, only creating it  if it doesn't already exist.</p>

===== Syntax =====
<a href="../Joomla-Framework/Registry/JRegistry.html">object JRegistry</a> &getConfig	( **$file**, **$type** )

| **$file** | string | The path to the configuration file |
| **$type** | string | The type of the configuration file |
