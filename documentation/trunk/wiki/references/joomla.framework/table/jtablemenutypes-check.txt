====== check ======
{#JAPI Joomla.Framework Session JTableMenutypes::check #}

Performs data integrity verification on the data. The method rejects any instances of "\'" character in the menutype field. If the title contains only whitespace then the value of the menutype field is assigned to it.

===== Syntax =====
boolean check ()


===== Examples =====
<code php|Example>
$menutype =& JTable::getInstance( 'menutypes' );
$menutype->menutype = 'custommen\'u';
$menutype->title = "     ";
$menutype->description = "This is a description of a custom menu.";

echo "Data Integrity Check " . ($menutype->check() ? "Passed" : "Failed -". $menutype->getError() . "<br />");
$menutype->menutype = 'custommenu';
echo $menutype->menutype . '<br />';

echo "Data Integrity Check " . ($menutype->check() ? "Passed" : "Failed -". $menutype->getError() . "<br />");
echo '<br />' . $menutype->title;
</code>
might produce:
<code php|Result>
Data Integrity Check Failed -The menu name cannot contain a \'
custommenu
Data Integrity Check Passed
custommenu
</code>
