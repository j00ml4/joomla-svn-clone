Gets the internal primary key name.

The primary key name is set when the constructor is called.

===== Syntax =====
string getKeyName ()


===== Examples =====
<code php|Example>
$section = JTable::getInstance( 'section' );

echo 'The name of the primary key is '.$section->getKeyName();
</code>
might produce
<code php|Result>
The name of the primary key is id
</code>
