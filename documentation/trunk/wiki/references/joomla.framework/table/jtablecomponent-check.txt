Performs data integrity verification on the data. Namely, it converts the parent and ordering properties to integers, if they are not already (this is done using the intval method).

===== Syntax =====
void check ()

===== Examples =====
<code php|Example>
$component =& JTable::getInstance('component' );
$component->loadByOption( 'com_document' );
$component->ordering = '5test';
echo "Data Integrity Check " . ($component->check() ? "Passed" : "Failed") . "<br />";
echo "The value of ordering is: $component->ordering <br />";

$component->parent = '3.0004';
echo "Data Integrity Check " . ($component->check() ? "Passed" : "Failed") . "<br />";
echo "The value of parent is: $component->parent <br />";
</code>
might produce:
<code php|Result>
Data Integrity Check Passed
The value of ordering is: 5
Data Integrity Check Passed
The value of parent is: 3
</code>
