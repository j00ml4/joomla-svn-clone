====== update ======
{#JAPI Joomla.Framework Session JTableSession::update #}

Updates the time of the current session. This will set the time field of the session to the current time. The $updateNulls parameter currently has no effect.

===== Syntax =====
void update	( **$updateNulls** )

| **$updateNulls** | boolean | is a boolean that specifies whether or not null values should be updated. This parameter currently has no effect. This parameter is optional and if omitted defaults to false. |

===== Examples =====
<code php|Example>
$session =& JTable::getInstance( 'session' );

$session->load( 'd799af5e42df6fba1aa3068406b6aa82' );
$session->update();
</code>
will update the time of the session with ID 'd799af5e42df6fba1aa3068406b6aa82'.
