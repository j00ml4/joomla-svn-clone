====== check ======
{#JAPI Joomla.Framework Session JTableModule::check #}

Performs data integrity verification on the data. It ensures that the module has a valid title.

===== Syntax =====
boolean check ()


===== Examples =====
<code php|Example>
$menutype =& JTable::getInstance( 'module' );
$menutype->title = "     ";

echo "Data Integrity Check " . ($menutype->check() ? "Passed" : "Failed -". $menutype->getError() . "<br />");
$menutype->title = 'My Module';

echo "Data Integrity Check " . ($menutype->check() ? "Passed" : "Failed -". $menutype->getError() . "<br />");
</code>
might produce:
<code php|Result>
Data Integrity Check Failed -Your Module must contain a title.
Data Integrity Check Passed
</code>
