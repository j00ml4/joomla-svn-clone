====== loadByOption ======
{#JAPI Joomla.Framework Session JTableComponent::loadByOption #}

Loads a data row using the specified option.

===== Syntax =====
boolean loadByOption ( **$option** )

| **$option** | string | is a string containing the component option value. |

===== Examples =====
<code php|Example>
$component =& JTable::getInstance( 'component' );
$component->loadByOption( 'com_document' );
echo "The component name is " . $component->name . ".";
</code>
might produce:
<code php|Result>
The component name is Document.
</code>
