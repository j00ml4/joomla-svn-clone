====== delete ======
{#JAPI Joomla.Framework Session JTableUser::delete #}

Deletes the record from the users table, deleting also any necessary ACL entries and message entries.

===== Syntax =====
void delete	( **$oid** )

| **$oid** | integer | is an integer containing the ID of the user to delete. |

===== Examples =====
<code php|Example>
$user =& JTable::getInstance( 'user' );

$user->load( 63 );

$user->delete();
</code>
will delete the user with an ID of 63.
