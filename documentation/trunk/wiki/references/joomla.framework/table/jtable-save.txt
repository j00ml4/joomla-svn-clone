Generic save method.

This method is a wrapper method that invokes other lower level methods of this class. This method will first bind() to the specified source array.  Then, it will invoke the check() method to ensure that the data is proper.  Then it will invoke the store() method to save the changes to the database.  Then it will invoke the checkin() method to update the record's checked_out field.  Finally, the reorder method will be invoked to compact the ordering values of the records in the table.

This method returns true on success, false otherwise.  Error messages are passed back using the classes built-in error handling.

===== Syntax =====
boolean save ( **$source**, **$order_filter** )
| **$source** | array | is an array containing the values to save to the object. |
| **$order_filter** | string | is a filter to use for the order updating. See the reorder() method for more details. This parameter is optional and if omitted defaults to an empty string and all records will be reordered. |


===== Examples =====

<code php|Example>
JTable::addIncludePath( JPATH_ADMINISTRATOR.DS.'components'.DS.'com_poll'.DS.'tables' );

$poll =& JTable::getInstance( 'poll', 'Table' );

$polldata = array(	'title' => 'sample poll',
					'alias' => 'sample',
					'voters' => 7,
					'published' => 1,
					'access' => 0,
					'lag' => 6000
				);
$poll->save( $polldata );

print_r( $poll );
</code>
