====== destroy ======
{#JAPI Joomla.Framework Session JTableSession::destroy #}

Destroys the pesisting session. Removes the record for the session from the table.

===== Syntax =====
void destroy ()

===== Examples =====
<code php|Example>
$session =& JTable::getInstance( 'session' );
$session->load( 'd799af5e42df6fba1aa3068406b6aa82' );

$session->destroy();
</code>
will result in the session with ID d799af5e42df6fba1aa3068406b6aa82 being destroyed.
