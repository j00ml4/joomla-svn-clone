Object constructor to set table and key field.

This method can be overloaded/supplemented by the child class.

===== Syntax =====
JTable <nowiki>__construct</nowiki> ( **$table**, **$key**, **&$db** )
| **$table** | string | is a string containing the name of the table in the db schema relating to the child class. |
| **$key** | string | is a string containing name of the primary key field in the table. |
| **&$db** | JDatabase | is a reference to the JDatabase object to use to access the database. |


===== Examples =====
This constructor is usually called from the constructor of a derived class. The constructor from the JTableComponent class is shown below as an example:
<code php|Example>
function __construct( &$db ) {
	parent::__construct( '#__components', 'id', $db );
}
</code>
