====== setLastVisit ======
{#JAPI Joomla.Framework Session JTableUser::setLastVisit #}

Updates the last visit time of user.

===== Syntax =====
boolean setLastVisit ( **$timeStamp**, **$id** )

| **$timeStamp** | integer | is an integer containing the timestamp to assign to the last visit field. This parameter is optional and if omitted defaults to the current timestamp. |
| **$id** | integer | is an integer containing the ID of the user to update. |

===== Examples =====
<code php|Example>
$user =& JTable::getInstance( 'user' );

$user->setLastVisit( mktime() - 900, 62 );
</code>
sets user 62's last vist date to fifteen minutes ago.
