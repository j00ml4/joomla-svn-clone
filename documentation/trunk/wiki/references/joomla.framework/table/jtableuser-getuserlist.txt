====== getUserList ======
{#JAPI Joomla.Framework Session JTableUser::getUserList #}

Returns a complete user list.

===== Syntax =====
array getUserList ()

===== Examples =====
<code php|Example>
$user =& JTable::getInstance( 'user' );

print_r( $user->getUserList() );
</code>
might produce:
<code php|Result>
Array ( [0] => Array ( [username] => admin ) )
</code>
