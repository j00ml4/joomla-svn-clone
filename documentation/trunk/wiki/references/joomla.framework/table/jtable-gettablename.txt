====== getTableName ======
{#JAPI Joomla.Framework Session JTable::getTableName #}

Gets the internal table name for the object.

===== Syntax =====
string getTableName ()


===== Examples =====

<code php|Example>
$category =& JTable::getInstance( 'category' );

echo 'The name of the table is '.$category->getTableName().'.';
</code>
might produce
<code php|Result>
The name of the table is #__categories.
</code>
