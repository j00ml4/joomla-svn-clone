====== check ======

Performs data integrity verification on the data. It will set fulltext to an empty string if it contains only &nbsp; or whitespace characters.

===== Syntax =====
void check ()

===== Examples =====
<code php|Example>
$content =& JTable::getInstance( 'content' );
$content->load( 6 );
$content->fulltext = "       &nbsp;&nbsp;  &nbsp;    \n\r";
echo "The value of fulltext is: '$content->fulltext' <br />";
echo "Data Integrity Check " . ($content->check() ? "Passed" : "Failed") . "<br />";
echo "The value of fulltext is: '$content->fulltext' <br />";
</code>
might produce:
<code php|Result>
The value of fulltext is: '      '
Data Integrity Check Passed
The value of fulltext is: '' 
</code>
