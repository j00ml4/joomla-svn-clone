====== getDBO ======
{#JAPI Joomla.Framework Session JTable::getDBO #}

Gets the internal database object.

This method returns a reference to the database object that the current JTable class is using to access the database.

This method will not work if called statically.

===== Syntax =====
JDatabase &getDBO ()

===== Examples =====
<code php|Example>
$content = JTable::getInstance( 'content' );

$db =& $content->getDBO();

$query = "SELECT * FROM #__content WHERE id = 7";
$db->setQuery( $query );
$row = $db->loadObject();

echo 'Title: ' . $row->title . '<br />';
</code>
might produce
<code php|Result>
Title: Example News Item 2
</code>
