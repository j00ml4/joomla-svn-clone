====== ipToNetAddress ======
{#JAPI Joomla.Framework Client JLDAP::ipToNetAddress #}

Converts a dot notation IP address to net address (e.g. for Netware, etc)

===== Syntax =====
string ipToNetAddress	( **$ip** )

| **$ip** | string | IP Address (e.g. xxx.xxx.xxx.xxx) |
