====== JFolder ======
{#JAPI Joomla.Framework FileSystem JFolder #}

This is a folder handling class. It provides an interface to the file system to perform various folder related functions. This is a [[http://php.net/manual/language.oop5.static.php|static class]] and so its methods are available without needing an instantiation of the class.
