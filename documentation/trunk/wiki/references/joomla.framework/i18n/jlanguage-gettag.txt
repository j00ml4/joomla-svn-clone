====== getTag ======
{#JAPI Joomla.Framework I18N JLanguage::getTag #}

Gets the RFC 3066 language tag.

===== Syntax =====
string getTag ()

===== Examples =====
<code php|Example>
$english = JLanguage::getInstance( 'en-GB' );
echo $english->getTag();
</code>
might produce:
<code php|Result>
en-GB
</code>
