====== ucwords ======

Makes a each word in the string's first character uppercase. See [[http://www.php.net/manual/en/function.ucwords.php|ucwords]] for more information.

===== Syntax =====
string ucwords ( **$str** )

| **$str** | string | is the string to convert the first letter of each word to uppercase. |

===== Examples =====
<code php|Example>
$string = "και εισηλθεν παλιν εις";
echo "If we ucwords $string we get: " . JString::ucwords( $string ) . ".";
</code>
might produce:
<code php|Result>
If we ucwords και εισηλθεν παλιν εις we get: Και Εισηλθεν Παλιν Εις.
</code>
