====== _ ======
{#JAPI Joomla.Framework I18N JLanguage::_ #}

Translator function, mimics the php gettext (alias _) function

===== Syntax =====
string _	( **$string**, **$jsSafe** )

| **$string** | string | The string to translate |
| **$jsSafe** | boolean | Make the result javascript safe |
