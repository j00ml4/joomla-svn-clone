====== getLocale ======
{#JAPI Joomla.Framework I18N JLanguage::getLocale #}

Gets the value of the locale property.

===== Syntax =====
string getLocale ()

===== Examples =====
<code php|Example>
$german = JLanguage::getInstance( 'de-DE' );
print_r( $german->getLocale() );
</code>
might produce:
<code php|Result>
Array ( [0] => ger [1] => de [2] => ger_DE [3] => de_DE [4] => german [5] => german-de [6] => germany )
</code>
