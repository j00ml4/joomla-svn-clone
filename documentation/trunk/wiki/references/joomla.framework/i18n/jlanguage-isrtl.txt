====== isRTL ======
{#JAPI Joomla.Framework I18N JLanguageHelper::isRTL #}

Gets the RTL property (i.e. determines of the language uses a script that moves from right to left).

===== Syntax =====
boolean isRTL	( **$value** )

| **$value** | string | is not used. |

===== Examples =====
<code php|Example>
$german = JLanguage::getInstance( 'de-DE' );
$hebrew = JLanguage::getInstance( 'he-IL' );
echo "German " . ($german->isRTL() ? 'is' : 'is not') . " a right to left language.";
echo "Hebrew " . ($hebrew->isRTL() ? 'is' : 'is not') . " a right to left language.";
</code>
might produce:
<code php|Result>
German is not a right to left language.Hebrew is a right to left language.
</code>
