Wrapper for the [[http://php.net/manual/function.sprintf.php|sprintf]] method. Similar to [[http://php.net/manual/function.sprintf.php|sprintf]], this method takes multiple parameters. See the [[http://php.net/manual/function.sprintf.php|PHP documentation] for information on parameters for this method. The first parameter given is the format string. This string is passed through the [[JText-_|JText::_]] method for translation before being passed on to [[http://php.net/manual/function.sprintf.php|sprintf]].

[[http://php.net/manual/function.printf.php|printf]] processes the string and returns the output as a string.


===== Syntax =====
string sprintf ( **$format**, **$args** )

| **$format** | string | is a string containing the format string to be processed. |
| **$args** | mixed | is a mixed array of parameters to pass to [[http://php.net/manual/function.sprintf.php|sprintf]]. See [[http://php.net/manual/function.sprintf.php|sprintf]] for more information. |

===== Examples =====
<code php|Example>
$output = JText::sprintf ('Characters: %c %c', 76, 65);
echo $output;
</code>
might produce
<code php|Result>
Characters: L A
</code>
