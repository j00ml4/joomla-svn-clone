====== setDebug ======
{#JAPI Joomla.Framework I18N JLanguage::setDebug #}

Sets the debug property. If the debug property is set, the _ method will add markers before and after each string to indicate whether or not the key exists in the language file.

===== Syntax =====
void setDebug	( **$debug** )

| **$debug** | boolean | is a boolean value to apply to the debug property. |

===== Examples =====
<code php|Example>
$german = JLanguage::getInstance( 'de-DE' );
echo "Debugging is " . ($german->getDebug() ? "on" : "off") . "<br />";
$german->setDebug( true );
echo "Turning debugging on...<br />";
echo "Debugging is " . ($german->getDebug() ? "on" : "off");
</code>
might produce:
<code php|Result>
Debugging is off
Turning debugging on...
Debugging is on
</code>
