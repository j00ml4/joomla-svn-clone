Builds a HTML iFrame for the specified URL.

===== Syntax =====
void Iframe (string $url, string $name, [array $attribs = null], integer $ssl)
| **$url** | string | The relative URL to use for the src attribute |
| **$name** | string  | The target attribute to use|
| **$attribs** | array| An associative array of attributes to add |
| **$ssl** | integer |Set the SSL functionality|
