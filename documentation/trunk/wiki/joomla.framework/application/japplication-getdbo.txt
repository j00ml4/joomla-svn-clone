====== getDBO ======

Return a reference to a JDatabase object instance. This method is the means by which extensions should obtain access to the Joomla! database object.

===== Syntax =====
JDatabase &getDBO ()

===== Examples =====
<code php|Example>
global $mainframe;

$database = $mainframe->getDBO();    
</code>
will result in a reference to the application database object being stored in the $database variable.
