====== getItemCount ======

Count the items in the menu for a certain type.

===== Syntax =====
integer getItemCount ( **$type** )

| **$type** | string | is a string containing the type of menu item to count. |

===== Examples =====
<code php|Example>
echo JApplicationHelper::getItemCount( 'content_blog_section' );
</code>
might produce
<code php|Result>
2
</code>
