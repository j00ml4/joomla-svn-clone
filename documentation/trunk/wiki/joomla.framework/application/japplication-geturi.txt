====== getURI ======

Return a reference to the JURI object. This method should be used by extensions to obtain access to information about the current request URI.

===== Syntax =====
JURI &getURI ()

===== Examples =====
<code php|Example>
global $mainframe;

$uri = $mainframe->getURI();
</code>
will result in a reference to the application's JURI object being stored in the $uri variable.
