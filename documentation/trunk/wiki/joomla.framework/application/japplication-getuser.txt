====== getUser ======

Return a reference to the JUser object. This method should be used by extensions to obtain access to information about the current request user.

===== Syntax =====
JUser &getUser ()

===== Examples =====
<code php|Example>
global $mainframe;

$urer = $mainframe->getUser();
</code>
will result in a reference to the application's JUser object being stored in the $user variable.
