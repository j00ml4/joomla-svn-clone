Returns the number of items in the menu that are of type 'blog section'.  This method is deprecated in Joomla! 1.5 (JApplicationHelper->getItemCount should be used instead).

===== Syntax =====
void getBlogSectionCount ()


===== Examples =====
<code php|Example>
global $mainframe;

echo $mainframe->getBlogSectionCount();
</code>
will display the number of menu items that are of type 'blog section'.
