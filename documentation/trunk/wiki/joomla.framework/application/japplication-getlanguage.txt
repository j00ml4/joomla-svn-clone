====== getLanguage ======

| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#Application|Application]] | {{http://api.joomla.org/media/images/Class.png?nolink}}[[JApplication|JApplication]] | [[http://api.joomla.org/Joomla-Framework/Application/JApplication.html|JApplication->getLanguage]] | never | Work in progress |

Return a reference to the JLanguage object. This method should be used by an extension to obtain access to the application's language object.

===== Syntax =====
JLanguage &getLanguage ()

===== Examples =====
<code php|Example>
global $mainframe;

$language = $mainframe->getLanguage();
</code>
would result in a reference to the application's language object being stored in the $language variable.
