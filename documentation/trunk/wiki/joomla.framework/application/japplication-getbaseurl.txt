Returns the base URL of the current application.

===== Syntax =====
string getBaseURL ()


===== Examples =====
<code php|Example>
global $mainframe;

echo $mainframe->getBaseURL()
</code>
might display the following:
<code php|Result>
http://127.0.0.1/joomla11/
</code>
if it is accessed from the site client.
