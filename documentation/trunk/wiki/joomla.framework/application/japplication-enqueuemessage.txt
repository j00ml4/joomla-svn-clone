====== enqueueMessage ======

Enqueues a message into the system message queue. The system messages are displayed when the page is rendered. The messages are rendered using <dt> and <dd> tags. The $type parameter is applied to the class of these tags. The messages are included in the template using <jdoc:include type="message" />.

===== Syntax =====
void enqueueMessage	( **$msg**, **$type** )

| **$msg** | string | is a string containing the message to enqueue. |
| **$type** | string | is a string containing the message type. This parameter is optional and if omitted defaults to 'message'. |

===== Examples =====
<code php|Example>
global $mainframe;

$mainframe->enqueueMessage( 'This is a test!' );
</code>
might result in:
<code php|Result>
<dl id="system-message" class="message fade">
<dt class="message">Message</dt>
<dd class="message">
	<ul>

		<li>This is a test!</li>
	</ul>
</dd>
</dl>
</code>
being rendered onto the page.
