====== getPathWay ======

| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#Application|Application]] | {{http://api.joomla.org/media/images/Class.png?nolink}}[[JApplication|JApplication]] | [[http://api.joomla.org/Joomla-Framework/Application/JApplication.html|JApplication->getPathWay]] | never | Work in progress |

Return a reference to the JPathWay object. This method should be used by extensions to obtain access to the pathway object.

===== Syntax =====
JPathway &getPathWay ()

===== Examples =====
<code php|Example>
global $mainframe;

$pathway = $mainframe->getPathWay();
</code>
will result in a reference to the application pathway object being stored in the $pathway variable.
