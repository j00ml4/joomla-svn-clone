====== getStaticContentCount ======

Returns the number of items in the menu that are of type 'static content'. This method is deprecated in Joomla! 1.5 (JApplicationHelper->getItemCount should be used instead).

===== Syntax =====
void getStaticContentCount ()

===== Examples =====
<code php|Example>
global $mainframe;

echo $mainframe->getStaticContentCount();
</code>
will display the number of menu items that are of type 'static content'.
