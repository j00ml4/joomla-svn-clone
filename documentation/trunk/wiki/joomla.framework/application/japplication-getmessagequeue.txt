====== getMessageQueue ======

Retrieves the system message queue. If the queue is empty, the message queue will be retrieved from the session.

===== Syntax =====
array getMessageQueue ()


===== Examples =====
<code php|Example>
global $mainframe;

$mainframe->enqueueMessage( 'The first message!' );
$mainframe->enqueueMessage( 'The second message!', 'alert' );
$mainframe->enqueueMessage( 'The third message!' );

print_r( $mainframe->getMessageQueue() );
</code>
might produce:
<code php|Result>
Array
(
    [0] => Array
        (
            [message] => The first message!
            [type] => message
        )

    [1] => Array
        (
            [message] => The second message!
            [type] => alert
        )

    [2] => Array
        (
            [message] => The third message!
            [type] => message
        )

)
</code>
