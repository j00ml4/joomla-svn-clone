====== renderButton ======

Renders a button and returns the HTML code. If the button type cannot be loaded, the method returns false. The first element in the passed array must be the button type. The other elements of the array should correspond to how they are handled by the button class (the node is passed to the [[JButton-render|render]] method).

===== Syntax =====
array renderButton ( **&$node** )

| **&$node** | array | is an array containing the type and the parameters to be passed to the button class. |

===== Examples =====
<code php|Example>
$bar =& new JToolBar( 'My ToolBar' );
$button =& $bar->loadButtonType( 'Standard' );
    
if ($button != false) {
    $params = array( 'Standard', 'send', 'Activate', 'activate', false );
    echo $bar->renderButton( $params );
}
</code>
might produce:
<code php|Result>
<td class="button" id="My ToolBar-send">

<a onclick="javascript:submitbutton('activate')" class="toolbar">
<div class="icon-32-send" title="Activate" type="Standard">
</div>
Activate
</a>
</td>
</code>
