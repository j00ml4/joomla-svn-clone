====== __construct ======

Constructs the JPagination object by initializing its properties to the specified values, making any necessary adjustments to ensure the validity of the properties. The $total parameter will be converted to an integer value. The $limitstart parameter will be set to 0 if it is negative. The $limit parameter will be set to 1 if it is less than 1. If $limit is more than the total number of records, $limitstart will be set to 0 (since all records will be displayed). 

===== Syntax =====
JPagination <nowiki>__construct</nowiki> ( **$total**, **$limitstart**, **$limit** )

| **$total** | integer | is an integer containing the total number of records controlled by the pagination list. |
| **$limitstart** | integer | is an integer containing the first record to display. |
| **$limit** | integer | is an integer containing the number of records to be displayed per page. |

===== Examples =====
The constructor is invoked in the following example:
<code php|Example>
global $mainframe, $option;
$db =& JFactory::getDBO();

$limit		= $mainframe->getUserStateFromRequest("$option.limit", 'limit', $mainframe->getCfg('list_limit'), 0);
$limitstart	= JRequest::getVar('limitstart', 0, '', 'int');

// get the total number of records
$query = "SELECT COUNT(*)"
. "\n FROM #__extcal_config"
;

$db->setQuery( $query );
$total = $db->loadResult();
jimport('joomla.presentation.pagination');
$pageNav = new JPagination( $total, $limitstart, $limit );

$query = "SELECT name, value"
. "\n FROM #__extcal_config"
;
$db->setQuery( $query, $pageNav->limitstart, $pageNav->limit );
$rows = $db->loadObjectList();
if ($db->getErrorNum()) {
	echo $db->stderr();
	return;
}

echo "<table>";
foreach ($rows as $row) {
    echo "<tr><td>$row->name</td><td>$row->value</td></tr>";
}
echo "</table>";
echo $pageNav->getPagesLinks( $mainframe->getBaseURL() . "index.php?option=$option" );
</code>
