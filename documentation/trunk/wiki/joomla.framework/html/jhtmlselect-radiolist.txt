====== radiolist ======

Generates an HTML radio list.

The specified options will be used to generate the list.  The option text can be translated by the method if desired.

The method returns a string containing the generated HTML code for the radio list.

===== Syntax =====
string radiolist	( **$arr**, **$tag_name**, **$tag_attribs**, **$selected**, **$key**, **$text**, **$idtag** )

| **$arr** | array | is an array of objects that are used to build the radio list.  The value and name pairs in the objects should correspond to the $key and $text parameters. |
| **$tag_name** | string | is a string containing the value of the HTML name attribute. |
| **$tag_attribs** | string | is a string containing additional HTML attributes for the tag. |
| **$key** | string | is a string containing the name of the object property for the option value. This parameter is optional and if omitted defaults to 'value'. |
| **$text** | string | is a string containing the name of the object property for the option text. This parameter is optional and if omitted defaults to 'text'. |
| **$selected** | mixed | is a variable containing the key that is to be selected in the list. |
| **$idtag** | string | is a string containing the value of the HTML id attribute. |
| **$translate** | boolean | is a boolean value which specifies whether or not the text should be translated using the JText::_() method. |

===== Examples =====
<code php|Example>
$db =& JFactory::getDBO();
$query = 'SELECT id AS value, title AS text FROM #__weblinks';
$db->setQuery( $query );
$weblinks = $db->loadObjectList();

echo JHTML::_( 'select.radiolist', $weblinks, 'weblink' );
</code>
