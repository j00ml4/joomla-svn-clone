====== writePagesCounter ======

Writes (echoes) the output of [[JPagination-getPagesCounter|getPagesCounter]]. This method has been deprecated as of Joomla! 1.5. Use [[JPagination-getPagesCounter|getPagesCounter]] instead.

===== Syntax =====
void writePagesCounter ()

===== Examples =====
<code php|Example>
global $mainframe, $option;
$db =& JFactory::getDBO();

$limit		= $mainframe->getUserStateFromRequest("$option.limit", 'limit', $mainframe->getCfg('list_limit'), 0);
$limitstart	= JRequest::getVar('limitstart', 0, '', 'int');
$query = "SELECT count(*) "
. "\n FROM #__weblinks"
;
$db->setQuery( $query );
$total = $db->loadResult();

jimport('joomla.presentation.pagination');
$pageNav = new JPagination( $total, $limitstart, $limit );
$pageNav->writePagesCounter();
</code>
might produce:
<code php|Result>
Results 1 - 5 of 5
</code>
