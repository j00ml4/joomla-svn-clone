Performs any necessary operations to construct the pane (this method in the abstract class does not do anything).

===== Syntax =====
JPane __construct ( **$params** )

| **$params** | array | is an associative array of values containing settings for the pane. See child classes for more information. This parameter is optional and if omitted defaults to an empty array. |

===== Examples =====
The [[JPane-getInstance|getInstance]] method calls the constructor in the following manner:
<code php|Example>
$instance = new $classname($params);
</code>
