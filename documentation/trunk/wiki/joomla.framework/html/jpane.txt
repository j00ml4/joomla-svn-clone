====== JPane ======

JPane is an abstract base class that provides an interface for panes. The term pane refers to the container and the term panel refers to the individual 'pages' inside the pane.

===== Methods =====
^ Method ^ Description ^
| [[JPane-__construct|{{http://api.joomla.org/media/images/Method.png }}]] [[JPane-__construct|__construct]] | Performs any necessary operations to construct the pane. |
| [[JPane-endPane|{{http://api.joomla.org/media/images/Method.png }}]] [[JPane-endPane|endPane]] | Outputs the necessary HTML code to end the pane. |
| [[JPane-endPanel|{{http://api.joomla.org/media/images/Method.png }}]] [[JPane-endPanel|endPanel]] | Outputs the necessary HTML code to end a panel. |
| [[JPane-getInstance|{{http://api.joomla.org/media/images/Method.png }}]] [[JPane-getInstance|getInstance]] | Returns a reference to a JPane object. |
| [[JPane-startPane|{{http://api.joomla.org/media/images/Method.png }}]] [[JPane-startPane|startPane]] | Outputs the necessary HTML code and JavaScript to start the pane. |
| [[JPane-startPanel|{{http://api.joomla.org/media/images/Method.png }}]] [[JPane-startPanel|startPanel]] | Outputs the necessary HTML code and JavaScript to create a panel with the title text and start the panel. |
