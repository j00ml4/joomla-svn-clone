JPaneSliders is a class used to draw parameter panes. This child class implements the JPane interface and produces a set of pages that can be changed using buttons that appear at the top of the pane. Pages will slide in and slide out. JPaneSliders look like:

JPaneSliders are based on the mooFX library which Joomla! provides built-in support for. Since this library uses JavaScript, it is necessary to have JavaScript installed in the browser in order for this to work.
