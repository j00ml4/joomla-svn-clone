====== JButton_Custom ======

Renders a custom button. This is an implimentation of the abstract [[JButton|JButton]] class. It provides the ability to create a fully customizable button. The HTML returned by its [[JButton_Custom-fetchButton|fetchButton]] method is supplied via a parameter.

===== Methods =====
^ Method ^ Description ^
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JButton_Custom-fetchButton|fetchButton]] | Returns the HTML code for the button. |
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JButton_Custom-fetchId|fetchId]] | Gets the button's CSS ID. |
