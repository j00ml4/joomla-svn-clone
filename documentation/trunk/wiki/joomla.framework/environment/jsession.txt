Class for managing HTTP sessions.

Provides access to session-state values as well as session-level settings and lifetime management methods. Based on the standard PHP session handling mechanism it provides for you more advanced features such as idle and expire timeouts and different session handler/container:
  * **file**: PHP's native file based session handler
  * **database**: database storage as [[references:joomla.framework:database:JDatabase|supported]] by Joomla!
  * **eAccelerator**: PHP accelerator, optimizer, and dynamic content cache [[http://eaccelerator.net]]
  * **APC**: Alternative PHP Cache, [[http://pecl.php.net/package/apc|PECL extension]] [[http://pecl4win.php.net/ext.php/php_apc.dll|win32]]
