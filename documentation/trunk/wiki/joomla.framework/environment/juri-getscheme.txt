====== getScheme ======

Gets the URI scheme (protocol - http, https, ftp, etc).
===== Syntax =====
string getScheme ()

===== Examples =====
<code php|Example>
$uri = new JURI( 'http://forum.joomla.org/index.php?topic=46226&action=new' );
echo "Scheme: " . $uri->getScheme();
</code>
might produce:
<code php|Result>
Scheme: http
</code>
