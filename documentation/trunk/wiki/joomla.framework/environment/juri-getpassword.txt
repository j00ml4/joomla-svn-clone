====== getPassword ======

| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#Environment|Environment]] | [[JURI|JURI]] | [[http://api.joomla.org/Joomla.Framework/Environment/JURI.html|JURI->getPassword]] |

Gets the URI password. This function returns the password, or null if no password was specified.
===== Syntax =====
string getPassword ()

===== Examples =====
<code php|Example>
$uri = new JURI( 'http://user:secret@www.joomla.org?action=help#create' );
echo "Password: " . $uri->getPassword();
</code>
might produce:
<code php|Result>
Password: secret
</code>
