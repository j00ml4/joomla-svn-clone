====== str_ireplace ======

Replaces one string with another in a case insensitive manner. See [[http://www.php.net/manual/en/function.str_ireplace.php|str_ireplace]] for more information.

===== Syntax =====
void str_ireplace	( **$search**, **$replace**, **$string**, **$count** )

| **$search** | string | is the existing string to replace |
| **$replace** | string | is the new string to replace with. |
| **$string** | string | is the string to search. |
| **$count** | integer | optional count value to be passed by referene |

===== Examples =====
<code php|Example>
$string = "Και Εισηλθεν Παλιν Εις";
$search = "ΠΑΛ";
$replace = "ΣΗΛ";
echo "When occurences of $search are replaced with $replace in the string $string in a case insensitive manner, we get " . JString::str_ireplace( $search, $replace, $string ) . ".";
</code>
might produce:
<code php|Result>
When occurences of ΠΑΛ are replaced with ΣΗΛ in the string Και Εισηλθεν Παλιν Εις in a case insensitive manner, we get Και Εισηλθεν ΣΗΛιν Εις.
</code>
