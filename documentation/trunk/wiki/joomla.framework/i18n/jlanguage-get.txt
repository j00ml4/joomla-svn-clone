Gets a metadata language property.

===== Syntax =====
mixed get ( **$property**, **$default** )

| **$property** | string | is a string containing the name of the property to retrieve. |
| **$default** | mixed | is a variable containing the default value to return if the property is not set. |

===== Examples =====
<code php|Example>
$german = JLanguage::getInstance( 'de-DE' );
echo $german->get( 'name' );
</code>
might produce:
<code php|Result>
German (Deutschland)
</code>
