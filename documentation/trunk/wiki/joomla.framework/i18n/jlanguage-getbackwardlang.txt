Gets the backward compatible language name.

===== Syntax =====
string getBackwardLang ()

===== Examples =====
<code php|Example>
$german = JLanguage::getInstance( 'de-DE' );
echo $german->getBackwardLang();
</code>
might produce:
<code php|Result>
german
</code>
