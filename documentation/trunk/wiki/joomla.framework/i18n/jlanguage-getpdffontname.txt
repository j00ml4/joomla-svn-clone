Gets the PDF font name to use for the language. This is used to properly render text in a PDF file to ensure that the proper font set is used.

===== Syntax =====
string getPdfFontName ()

===== Examples =====
<code php|Example>
$english = JLanguage::getInstance( 'en-GB' );
echo $english->getPDFFontName();
</code>
might produce:
<code php|Result>
vera
</code>
