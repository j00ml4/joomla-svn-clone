====== detectLanguage ======

| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#I18N|I18N]] | {{http://api.joomla.org/media/images/Class.png?nolink}}[[JLanguageHelper|JLanguageHelper]] | [[http://api.joomla.org/Joomla-Framework/I18N/JLanguageHelper.html|JLanguageHelper->detectLanguage]] | never | Work in progress |

Tries to detect the language. This is done by retrieving the value of $_SERVER['HTTP_ACCEPT_LANGUAGE'] and comparing these values to the list of known (installed) languages. If a match cannot be found, 'en-GB' is returned.

===== Syntax =====
void detectLanguage ()

===== Examples =====
<code php|Example>
echo JLanguageHelper::detectLanguage();
</code>
might produce:
<code php|Result>
en-GB
</code>
