Wrapper for the [[http://ca.php.net/manual/en/function.printf.php|printf]] method. Similar to [[http://ca.php.net/manual/en/function.printf.php|printf]], this method takes multiple parameters. See the [[http://ca.php.net/manual/en/function.printf.php|PHP documentation] for information on parameters for this method. The first parameter given is the format string. This string is passed through the [[JText-_|JText::_]] method for translation before being passed on to [[http://ca.php.net/manual/en/function.printf.php|printf]].

[[http://ca.php.net/manual/en/function.printf.php|printf]] processes the string and prints output to the standard output. This method returns the length of the outputted string.

===== Syntax =====
integer printf ( **$format**, **$args** )

| **$format** | string | is a string containing the format string to be printed. |
| **$args** | mixed | is a mixed array of parameters to pass to printf. See [[http://ca.php.net/manual/en/function.printf.php|printf]] for more information. |

===== Examples =====
<code php|Example>
JText::printf ('Characters: %c %c', 76, 65);
</code>
might output to the display
<code php|Result>
Characters: L A
</code>
