====== createLanguageList ======

Builds a list of the system languages which can be used in a select option. The results are returned in an array of associate arrays with three fields: text, value, and selected.

===== Syntax =====
void createLanguageList	( **$actualLanguage**, **$basePath** )

| **$actualLanguage** | string | is a string containing the language that should be selected. |
| **$basePath** | string | is a string containing the base path to use. This parameter is optional and if omitted defaults to JPATH_BASE. |

===== Examples =====
<code php|Example>
$language =& JFactory::getLanguage();
$languages = JLanguageHelper::createLanguageList( $language->getTag(), JPATH_BASE . DS . 'installation' );
print_r( $languages );
</code>
