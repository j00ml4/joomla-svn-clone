Gets the name of the language.

===== Syntax =====
string getName ( **$value** )

| **$value** | string | is an optional value that has no effect. |

===== Examples =====
<code php|Example>
$german = JLanguage::getInstance( 'de-DE' );
echo $german->getName();
</code>
might produce:
<code php|Result>
German (Deutschland)
</code>
