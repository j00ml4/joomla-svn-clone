====== hasKey ======

Determines if a key exists in the language string array.

===== Syntax =====
boolean hasKey ( **$key** )

| **$key** | string | is a string containing the key to check. |

===== Examples =====
<code php|Example>
$german = JLanguage::getInstance( 'de-DE' );
$key1 = "ADMIN";
$key2 = "ADMEN";
   
echo "The key $key1 " . ($german->hasKey( $key1 ) ? "exists" : "does not exist") . " in the language file.";
echo "The key $key2 " . ($german->hasKey( $key2 ) ? "exists" : "does not exist") . " in the language file.";
</code>
might produce:
<code php|Result>
The key ADMIN exists in the language file.The key ADMEN does not exist in the language file.
</code>
