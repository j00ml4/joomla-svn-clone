Determines whether there is an authenticated user in the current session.

===== Syntax =====
boolean isAuthenticated ()

===== Examples =====
<code php|Example>
jimport( 'joomla.application.user.authenticate' );

echo (JAuthenticateHelper::isAuthenticated()) ? "We're in!" : "No go.";
</code>
might produce:
<code php|Result>
No go.
</code>
