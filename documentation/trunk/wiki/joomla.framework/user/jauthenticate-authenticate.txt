====== authenticate ======

Finds out if a set of login credentials are valid by asking all observing objects to run their respective authentication routines. If authentication is successful, the function returns the userid of the authenticated user. If it fails, the function returns false.

===== Syntax =====
mixed authenticate ( **$credentials** )
| **$credentials** | array | is an associative array containing the credentials to authenticate. The username should be stored in $credentials['username'] and the password should be stored in $credentials['password']. |

===== Examples =====
<code php|Example>
jimport( 'joomla.application.user.authenticate' );
$authenticate = JAuthenticate::getInstance();
$userid = $authenticate->authenticate( array( 'username' => 'user1', 'password' => 'secret' ) );

if ($userid) {
    echo "User authentication successful: User ID: " . $userid;
} else {
    echo "Authentication failed!";
}
</code>
