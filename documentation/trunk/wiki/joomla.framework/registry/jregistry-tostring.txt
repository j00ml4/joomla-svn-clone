====== toString ======

Transforms a namespace to a specified string format. If the namespace is not specified, the default namespace is used.

===== Syntax =====
string toString	( **$format**, **$namespace**, **$params** )

| **$format** | string | is a string specifying the format in which to return the string. This can be 'INI', 'XML' or 'PHP'. |
| **$namespace** | string | is a string specifying the namespace to return. This parameter is optional and if omitted defaults to the default namespace. |
| **$params** | array | is an associative array of parameters that are used by the formatter. See the [[JRegistryFormat|JRegistryFormat]] classes for more information. |

===== Examples =====
<code php|Example>
$registry = new JRegistry( 'com_document' );
$registry->setValue( 'joomla.content.showauthor', true );
$registry->setValue( 'joomla.debug', 'yes' );
print_r( $registry->toString( 'XML', 'joomla' ) );
</code>
might produce:
<code php|Result>
<?xml version="1.0" ?>

<config>
	<group name="content">
		<entry name="showauthor">1</entry>
	</group>
	<entry name="debug">yes</entry>
</config>
</code>
