====== loadINI ======

Loads an INI formatted string into the registry. If the namespace is not specified, the data will be loaded into the default namespace.

===== Syntax =====
boolean loadINI	( **$data**, **$namespace** )

| **$data** | string | is a string containing the INI data to load. |
| **$namespace** | string | is a string specifying the namespace into which to load the data. This parameter is optional and if omitted defaults to the default namespace. |

===== Examples =====
<code php|Example>
$registry = new JRegistry( 'com_document' );
jimport('joomla.filesystem.file');
$data = JFile::read( JPATH_SITE . DS . 'language' . DS . 'en-GB' . DS . 'en-GB.ini' );
$registry->loadINI( $data );
echo "The registry value is: " . $registry->getValue( 'com_document.ADMINISTRATOR' );
</code>
might produce:
<code php|Result>
The registry value is: administrator
</code>
