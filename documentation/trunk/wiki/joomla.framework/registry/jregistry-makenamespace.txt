====== makeNameSpace ======

Creates a namespace with the specified name. Care should be taken when using this method because there is no check to see if the namespace already exists, and therefore data in an existing namespace could be lost.

===== Syntax =====
boolean makeNameSpace ( **$namespace** )

| **$namespace** | string | is a string containing the name of the namespace to create. |

===== Examples =====
<code php|Example>
$registry = new JRegistry( 'com_document' );
$registry->makeNameSpace( 'joomla' );

$data = array( 'name' => 'Bob', 'address' => '123 Yonge St.' );
$registry->loadArray( $data, 'joomla' );
print_r( $registry->_registry );
echo "The value is: " . $registry->getValue( 'joomla.name' );
</code>
might produce:
<code php|Result>
The registry value is: Bob
</code>
