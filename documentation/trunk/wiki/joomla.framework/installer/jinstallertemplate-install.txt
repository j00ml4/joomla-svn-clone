Custom install method for templates. The install method performs the following actions:
  * finds and checks the installation file and verifies that it is the proper installation type
  * determines the client application target
  * creates the template directory
  * copies files
  * creates template positions
  * copies setup file to the appropriate place

This method returns true on success, false otherwise. 

===== Syntax =====
boolean install	( **$p_fromdir** )

| **$p_fromdir** | string | is a string containing the directory from which to install the template |
