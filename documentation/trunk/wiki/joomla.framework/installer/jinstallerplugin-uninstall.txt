Custom uninstall method for plugins. The uninstall method performs the following actions:
  * determines the plugin directory
  * determines path to the XML install file
  * deletes plugin object
  * deletes files and removes directory
  
This method returns true on success, false otherwise. 

===== Syntax =====
boolean uninstall ( **$id**, **$client** )

| **$id** | integer | is an integer containing the id of the plugin to uninstall. |
| **$client** | integer | is an integer containing the client id. |
