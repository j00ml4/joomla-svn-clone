Gets a file name out of a URL. The passed parameter is split into parts using '/' as the separator and the final part is returned. If no valid result is found, false is returned.

===== Syntax =====
string getFilenameFromURL ( **$url** )

| **$url** | string | is a string containing the URL to get the filename from. |
