====== getTitle ======

Gets the title of the current document.
===== Syntax =====
string getTitle ()

===== Examples =====
This example displays the title of the current document.
<code php|Example>
global $mainframe;

$document = &JFactory::getDocument();
echo "Title: " . $document->getTitle();
</code>
Might produce:
<code php|Result>
Title: Joomla! 1.5 Documentation Test Bed
</code>
