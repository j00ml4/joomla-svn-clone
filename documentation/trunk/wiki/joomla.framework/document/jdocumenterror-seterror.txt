Sets the error object.  The error object should be an object of type JException or Exception (for PHP5).

===== Syntax =====
boolean setError	( **$error** )

| **$error** | object | is an object of type JException (or Exception) to set. |

===== Examples =====
<code php|Example>
$error =& JError::raiseError( 404, 'Not found' );
$errorDoc =& JDocument::getInstance( 'error' );
$errorDoc->setError( $error );
</code>
