====== fetchBody ======
^ API ^ Package ^ Class ^ phpDocumentor ^ Last reviewed ^ Doc status ^
| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#Document|Document]] | [[JDocument|JDocument]] | [[http://api.joomla.org/Joomla-Framework/Document/JDocument.html|JDocument->fetchBody]] |

This function is used to return the document body. It is an abstract function and can be overriden by a subclass to provide this functionality.
===== Syntax =====
string fetchBody ()

===== Examples =====
<code php|Example>
global $mainframe;

$document = &JFactory::getDocument();
$body = $document->fetchBody();
</code>
This function in the JDocument class returns an empty string.
