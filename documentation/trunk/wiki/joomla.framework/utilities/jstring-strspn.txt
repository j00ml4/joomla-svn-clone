====== strspn ======

Finds length of initial segment matching mask. See [[http://www.php.net/manual/en/function.strspn.php|strspn]] for more information.

===== Syntax =====
void strspn	( **$haystack**, **$mask**, **$start**, **$length** )

| **$haystack** | string | is a string containing the text to search. |
| **$mask** | string | is a string containing the characters that form the mask. |
| **$start** | integer | is an integer specifying the position at which to start searching. This parameter is optional and if omitted defaults to 0. |
| **$length** | integer | is an integer specifying the number of characters to search. This parameter is optional and if omitted the entire string will be searched. |

===== Examples =====
<code php|Example>
$haystack = "και εισηλθεν παλιν εις";
$mask = "ακσ ιε";
echo "The number of characters at the beginning of $haystack that contain the characters $mask is " . JString::strspn( $haystack, $mask ) . ".";
</code>
might produce:
<code php|Result>
The number of characters at the beginning of και εισηλθεν παλιν εις that contain the characters ακσ ιε is 7.
</code>
