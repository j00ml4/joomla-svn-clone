====== fromObject ======

Maps an object to an array. This method will recurse unless specified otherwise. An optional regular expression parameter allows field names to be filtered.

===== Syntax =====
array fromObject ( **$p_obj**, **$recurse**, **$regex** )

| **$p_obj** | object | is the source object. |
| **$recurse** | boolean | is a boolean value that is set to true to recurse through multi-level objects. This parameter is optional and if omitted defaults to true. |
| **$regex** | string | is a string containing the regular expression to match on field names. This parameter is optional and if omitted defaults to null. |

$example->length    = 25;
    
echo "The example object: ";
print_r ( $example );
echo "In array form: ";
print_r ( JArrayHelper::fromObject( $example, false ) );
