Finds the position of the last occurrence of a string. See [[http://php.net/manual/function.strrpos.php|strrpos]] for more information.

===== Syntax =====
mixed strrpos	( **$haystack**, **$needle** )

| **$haystack** | string | is the string being examined. |
| **$needle** | string | is the string being searced for. |

===== Examples =====
<code php|Example>
$haystack = "και εισηλθεν παλιν εις";
$needle = "α";
echo "The position of the last occurence of $needle in $haystack is " . JString::strrpos( $haystack, $needle ) . ".";
</code>
might produce:
<code php|Result>
The position of the last occurence of α in και εισηλθεν παλιν εις is 14.
</code>
