====== trim ======

Strip whitespace (or other characters) from the beginning and end of a string. Note: you only need to use this if you are supplying the optional charlist argument and it contains UTF-8 characters. Otherwise trim will work normally on a UTF-8 string. See [[http://www.php.net/manual/en/function.trim.php|trim]] for more information.

===== Syntax =====
string trim	( **$str**, **$charlist** )

| **$str** | string | is the string to be trimmed |
| **$charlist** | string | is a string containing the optional charlist of additional characters to trim. This parameter is optional and if omitted, defaults to the standard whitespace characters. If this parameter is omitted, this method will yield the same results as the built-in php [[http://www.php.net/manual/en/function.trim.php|trim]] function. |

===== Examples =====
<code php|Example>
$string = "τττττΚαι Εισηλθεν Παλιν Ειςτττττττ";
$trim = "Κτ";
echo "If we trim the $trim characters off the ends of $string we get: " . JString::trim( $string, $trim ) . ".";
</code>
might produce:
<code php|Result>
If we trim the Κτ characters off the ends of τττττΚαι Εισηλθεν Παλιν Ειςτττττττ we get: αι Εισηλθεν Παλιν Εις.
</code>
