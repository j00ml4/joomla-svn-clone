====== getPublicProperties ======

This function returns an array of public property names.

===== Syntax =====
array getPublicProperties ()

===== Examples =====
<code php|Example>
jimport( 'joomla.application.search' );
$search =& new JSearch();

print_r( $search->getPublicProperties() );
</code>
might produce:
<code php|Result>
Array
(
    [0] => text
    [1] => ordering
    [2] => matchType
    [3] => limitstart
    [4] => limit
    [5] => count
    [6] => results
)
</code>
