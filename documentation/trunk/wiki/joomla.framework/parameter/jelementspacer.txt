====== JElementSpacer ======

This class implements the JElement interface and is used to render a spacer element. The JElementSpacer class can be used to render a custom HTML spacer tag.

===== Methods =====
^ Method ^ Description ^
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JElementSpacer-fetchElement|fetchElement]] | Renders the HTML code for a spacer. |
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JElementSpacer-fetchTooltip|fetchTooltip]] | Renders the tooltip for the spacer. This method will always return '&nbsp;'. |
