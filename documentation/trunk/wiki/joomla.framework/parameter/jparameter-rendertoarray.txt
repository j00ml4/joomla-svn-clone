====== renderToArray ======
^ API ^ Package ^ Class ^ phpDocumentor ^ Last reviewed ^ Doc status ^
| {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework|Joomla.Framework]] | {{http://api.joomla.org/media/images/package.png?nolink}} [[references:joomla.framework/#Parameter|Parameter]] | {{http://api.joomla.org/media/images/Class.png?nolink}}[[JParameter|JParameter]] | [[http://api.joomla.org/Joomla-Framework/Parameter/JParameter.html|JParameter->renderToArray]] | never | Work in progress |

Renders the parameters into HTML. The parameters are rendered by the [[JElement|JElement]] subclasses if a setup file was found. Otherwise, the parameters are displayed in raw INI format inside a text area form element.

===== Syntax =====
array renderToArray	( **$name** )

| **$name** | string | is a string containing the name of the control, or of the default text area if a setup file is not found. This parameter is optional and if omitted defaults to 'params'. |

===== Examples =====
<code php|Example>
$paramString = "name=\n"
    . "position=Chef\n"
    . "category=4"
    ;

$xml = & JFactory::getXMLParser('Simple');
    
$file 	= JPATH_ADMINISTRATOR .'/components/com_document/document_items.xml';
$params = new JParameter( $paramString, $file );
print_r( $params->renderToArray() );
</code>
might produce:
