====== getParams ======

Renders each parameter and returns the results in an array. If no setup file was found, then the parameters are rendered as a TEXTAREA control containing the raw INI data.

===== Syntax =====
array getParams	( **$name** )

| **$name** | string | is a string containing the name of the form control or the name of the default text area (if the setup file was not found). This parameter is optional and if omitted defaults to 'params'. |

===== Examples =====
<code php|Example>
$paramString = "name=\n"
    . "position=Chef\n"
    . "category=4"
    ;

$xml = & JFactory::getXMLParser('Simple');
    
$file 	= JPATH_ADMINISTRATOR .'/components/com_document/document_items.xml';
$params = new JParameter( $paramString, $file );

print_r( $params->getParams() );
</code>
might produce:
