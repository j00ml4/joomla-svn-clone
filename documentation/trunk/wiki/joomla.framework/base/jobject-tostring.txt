Object-to-string conversion. This is function is empty and can be defined by subclasses.
===== Syntax =====
string toString ()

===== Examples =====
In the base class, this function returns the name of the class to which the object belongs.
<code php|Example>
$object = new JObject();
echo $object->toString();
</code>
Might produce:
<code php|Result>
jobject
</code>
