====== JFolder ======

This is a folder handling class. It provides an interface to the file system to perform various folder related functions. This is a [[http://www.php.net/manual/en/language.oop5.static.php|static class]] and so its methods are available without needing an instantiation of the class.

| [[JFolder-exists|{{http://api.joomla.org/media/images/Method.png }}]] [[JFolder-exists|exists]] | Wrapper for the standard file_exists function. |

| [[JFolder-move|{{http://api.joomla.org/media/images/Method.png }}]] [[JFolder-move|move]] | Moves a folder. |
