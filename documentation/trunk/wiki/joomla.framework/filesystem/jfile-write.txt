Write a string to a file. If the current user has permission and safe mode is turned off, then this function will use the PHP [[http://www.php.net/file-put-contents|file_put_contents]] function. Otherwise, the function will use FTP (if enabled) to write the file. The function returns true if the write operation was successful.  Otherwise, the function returns false and may raise a warning using the [[references:Joomla.Framework:utilities:JError|JError]] class.

===== Syntax =====
boolean write ( **$file**, **$buffer** )
| **$file** | string | is a string containing the full file path. |
| **$buffer** | string | is a string containing the data to be written to the file. |

===== Examples =====
Example: This example writes some text to a file.
<code php|Example>
$buffer = "Joomla! is one of the most powerful Open Source Content Management Systems on the planet. It is used all over the world for everything from simple websites to complex corporate applications. Joomla! is easy to install, simple to manage, and reliable. ";

$file = 'joomlablurb.txt';

JFile::write( $file, $buffer );
</code>
