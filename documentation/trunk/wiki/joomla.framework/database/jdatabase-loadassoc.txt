====== loadAssoc ======

Executes the current query and returns the first result row in an associative array.

===== Syntax =====
void loadAssoc ()

===== Examples =====
<code php|Example>
$dbo =& JFactory::getDBO();

$query = "SELECT name, clickurl FROM #__banner";
$dbo->setQuery( $query );
$row = $dbo->loadAssoc();
print_r( $row );
echo "<br />";
$row = $dbo->loadAssoc();
print_r( $row );
</code>
might produce:
<code php|Result>
Array ( [name] => OSM 1 [clickurl] => http://www.opensourcematters.org )
Array ( [name] => OSM 1 [clickurl] => http://www.opensourcematters.org )
</code>
Note that this method can only be used to obtain the first result (as illustrated above).
