====== getCollation ======

Returns the collation being used by the database. Collations are supported by MySQL database starting with version 4.1.2. Previous versions will return 'N/A (mySQL < 4.1.2)'.

===== Syntax =====
string getCollation ()


===== Examples =====
<code php|Example>
$dbo = JFactory::getDBO();
echo $dbo->getCollation();
</code>
might produce:
<code php|Result>
utf8_general_ci
</code>
