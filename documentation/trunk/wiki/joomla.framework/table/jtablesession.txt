====== JTableSession ======

The JTableSession class is an implementation of the JTable interface which provides access to the session table. This implement provides methods ti insert sessions, purge old sessions and to update sessions.

===== Methods =====
^ Method ^ Description ^
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JTableSession-__construct|__construct]] | calls the parent constructor to attach a JDatabase object to itself and to define its table name and primary key and initializes session variables. |
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JTableSession-destroy|destroy]] | destroys the persisting session by removing the record from the table. |
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JTableSession-insert|insert]] | inserts a new session into the table. |
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JTableSession-purge|purge]] | purges old sessions from the table. |
| {{http://api.joomla.org/media/images/Method.png?nolink}} [[JTableSession-update|update]] | updates the session time field. |

===== Properties =====
^ Property ^ Description ^
| **$session_id** | is a string containing a unique identifier for the session. |
| **$time** | is a string containing the time that the session was last accessed. |
| **$userid** | is an integer containing a reference to a user ID from the users table. |
| **$usertype** | is a string containing the usertype that belongs to the session. |
| **$username** | is a string containing the username of the session user. |
| **$gid** | is an integer containing the group ID of the session user. |
| **$guest** | is a boolean specifying whether or not the session user is a guest. |
| **$client_id** | is an integer specifying which client the user is accessing. |
