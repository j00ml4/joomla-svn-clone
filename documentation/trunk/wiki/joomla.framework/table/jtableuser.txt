The JTableUser class is an implementation of the JTable interface which provides access to the users table. This implement provides methods to handle parameters, data verification, user deletion, updating last visit information and storing a user. It also provides a method of obtaining a complete user list or a user list for a specific group.

===== Properties =====
^ Property ^ Description ^
| **$id** | is an integer that contains the primary key for the table. |
| **$name** | is a string containing the name of the user. |
| **$username** | is a string containing the username of the user. |
| **$email** | is a string containing the user's email address. |
| **$password** | is a string containing the user's password. |
| **$usertype** | is a string containing the user type. |
| **$block** | is a boolean specifying whether or not the user is blocked. |
| **$sendEmail** | is a boolean specifying whether or not email should be sent to the user. |
| **$gid** | is an integer containing the group ID of the user. |
| **$registerDate** | is a datetime containing the date the user registered. |
| **$lastVisitDate** | is a datetime containing the date of the user's last visit. |
| **$activiation** | is a string containing the activation code for the user. |
| **$params** | is a string containing the additional parameters for the user. |
